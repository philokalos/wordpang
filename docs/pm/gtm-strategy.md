# Go-To-Market Strategy — WordPang

**Date:** 2026-03-22
**출시 목표:** App Store 제출 → TestFlight 테스트 → 공개 출시
**예산:** $0 (유기적 성장 전략)

---

## 1. Market Selection — 비치헤드 세그먼트

**첫 번째 타겟: 교육 앱에 관심 높은 학부모 (30~40대) + 그 자녀 (초등 3~5학년)**

| 기준 | 평가 | 이유 |
|------|------|------|
| Pain 강도 | ★★★★★ | 구독 피로도 최고조, 광고·데이터 걱정 높음 |
| 접근 가능성 | ★★★★☆ | 네이버 맘카페, 육아 블로그 — 명확한 채널 |
| 시장 규모 | ★★★☆☆ | 한국 초등학생 약 250만, 스마트폰 보유 추산 150만 |
| 전환 가능성 | ★★★★★ | 무료·무광고 → 설치 장벽 없음 |
| 바이럴 잠재력 | ★★★★☆ | 맘카페 추천이 폭발적 확산 구조 |

## 2. 핵심 메시지 아키텍처

### 마스터 메시지
> **"외우지 말고 맞혀보세요 — 초등 영어 단어를 발견하는 게임"**

### 학부모 채널 메시지 (Primary)
> "우리 아이가 혼자서 영어 앱을 켜요. 광고 없음 · 개인정보 수집 없음 · 완전 무료"

### 아동 채널 메시지 (Secondary)
> "색깔이 정답을 알려줘요 🟩🟧🟫 — 게임 한 판에 단어 하나 완전 정복!"

### 교사 채널 메시지 (Future)
> "교육 연구 기반 어휘 학습. 간격 반복으로 장기 기억 전환. 무료로 교실에서 활용하세요."

### 메시지 계층

```
Layer 1 (Why): 아이가 영어를 외우지 않고 발견하는 경험
Layer 2 (How): 색깔 힌트 퍼즐 + 과학적 간격 반복 + 무광고·무데이터
Layer 3 (What): 514개 단어 × 8카테고리 × 21개 배지 × 완전 오프라인
```

## 3. 채널 전략

### Phase 1 채널 (출시 전~출시 후 30일)

#### App Store 최적화 (ASO) — P0
| 항목 | 실행 내용 |
|------|---------|
| 앱 이름 | WordPang — 초등 영어 단어 게임 |
| 부제목 | 퍼즐로 배우는 초등 영단어 |
| 키워드 | 영어단어,초등영어,단어게임,영어퍼즐,워드팡,단어학습,어휘력,복습,플래시카드,영단어퀴즈 |
| 스크린샷 순서 | 1.게임플레이 → 2.색깔힌트 → 3.힌트패널 → 4.학습카드 → 5.복습모드 → 6.배지 → 7.통계 |
| 프로모션 텍스트 | 출시 시즌 키워드 반영 (예: "새학기 영어 준비") |

#### 네이버 맘카페 · 육아 커뮤니티 — P0
| 항목 | 실행 내용 |
|------|---------|
| 타겟 커뮤니티 | 맘카페 (교육), 네이버 카페 "초등영어", 아이엠스쿨 |
| 콘텐츠 전략 | 진솔한 개발 스토리 + "왜 만들었나" 글 → 스팸 아닌 가치 공유 |
| 메시지 | "아이 영어 앱 찾다가 직접 만들었어요 — 완전 무료, 광고 없음, 개인정보 수집 없음" |
| 게시 타이밍 | 새학기(3월), 방학(7~8월), 개학(9월) 맞춤 |

#### TestFlight 베타 — P0 (출시 전)
| 항목 | 내용 |
|------|------|
| 베타 테스터 | 지인 학부모 5~10명 + 그 자녀 |
| 목표 | 첫 게임 완료율, 학습 카드 확인율, 크래시 발견 |
| 피드백 수집 | 수동 인터뷰 (5~10분) |

### Phase 2 채널 (출시 후 30~90일)

#### 결과 공유 바이럴 — P1
| 항목 | 내용 |
|------|------|
| 공유 포맷 | 🟩🟧⬜ 이모지 결과 그리드 + "WordPang" 텍스트 |
| 플랫폼 | 인스타그램 스토리, 카카오톡, 트위터/X |
| 촉진 방법 | 결과 화면에 공유 버튼 prominence 높이기 |

#### 인스타그램 교육 계정 — P1
| 항목 | 내용 |
|------|------|
| 타겟 계정 | "엄마표영어", "초등영어", "육아" 해시태그 계정 |
| 접근 방식 | DM으로 무료 앱 소개, 리뷰 요청 |
| 콘텐츠 소재 | 이모지 공유 결과, 배지 달성 화면, 통계 화면 |

### Phase 3 채널 (출시 후 90일+)

#### 교사 채널 (B2B2C) — P2
- 초등 영어 교사 커뮤니티 (인디스쿨, 선생님닷컴) 접촉
- 교실 활용 가이드 1페이지 PDF 제작
- "수업 5분 warm-up으로 WordPang" 사용법 공유

#### YouTube 교육 채널 협업 — P2
- 엄마표 영어, 초등 영어 채널 협업 요청
- 리뷰 영상 or 짧은 소개 영상 제작

## 4. Launch Plan (90일 로드맵)

### Pre-Launch (D-30 ~ D-1)

| 주차 | 활동 |
|------|------|
| D-30 | TestFlight 베타 10명 배포, 크래시 수집 |
| D-21 | v3.2 발음 오디오 완료 (가능 시) |
| D-14 | App Store 메타데이터 최종 작성 (스크린샷, 설명, 키워드) |
| D-7 | App Store 제출 → 심사 (평균 24~48시간) |
| D-3 | 맘카페 사전 소개 글 작성 준비 |
| D-1 | 앱 승인 확인, 출시 버튼 준비 |

### Launch Week (D-Day ~ D+7)

| 일자 | 활동 |
|------|------|
| D+0 | App Store 공개 출시 |
| D+0 | 네이버 맘카페 소개 글 2~3곳 게시 |
| D+1 | 인스타그램 교육 계정 DM 10~20곳 |
| D+3 | 지인·베타 테스터에게 리뷰 요청 (App Store 첫 리뷰 확보) |
| D+7 | 첫 주 지표 점검 (다운로드, 크래시율, 리뷰) |

### Post-Launch (D+8 ~ D+90)

| 기간 | 활동 |
|------|------|
| D+8~30 | 리뷰 응답, ASO 키워드 조정, D3/D7 리텐션 측정 |
| D+31~60 | 바이럴 공유 기능 개선, 맘카페 2차 게시 |
| D+61~90 | 교사 채널 소규모 테스트, v3.2 오디오 출시 (미완성 시) |

## 6. Go/No-Go Decision Points

| 체크포인트 | 시점 | 기준 |
|---------|------|------|
| TestFlight 통과 | 출시 7일 전 | 크래시율 < 0.5%, 완료율 > 60% |
| 출시 30일 검토 | D+30 | 다운로드 > 200, 평점 ≥ 4.0★ |
| 전략 재검토 | D+90 | D3 리텐션 < 15% → 온보딩 개선 / North Star < 2 → 핵심 루프 재설계 |

## 7. Risk Mitigation

| 리스크 | 확률 | 대응 |
|--------|------|------|
| App Store 심사 거부 | 중간 | 심사 노트 사전 준비, 데모 계정 불필요 명시 |
| 출시 후 낮은 다운로드 | 높음 | ASO 즉시 A/B 조정, 맘카페 2차 게시 |
| 크래시 급증 | 낮음 | TestFlight 충분히 검증 후 출시 |
| 경쟁사 유사 기능 출시 | 낮음 | 무데이터·무광고 포지션은 즉시 대응 불가 |
