# SWOT Analysis — WordPang

**Date:** 2026-03-22
**Context:** App Store 출시 준비 단계 (v3.1.2)

---

## Strengths (내부 강점)

| # | 강점 | 설명 |
|---|------|------|
| S1 | **교육 연구 기반 설계** | 4개 검증된 이론(교정 피드백·간격 반복·Flow Zone·역량 동기) 통합 — 단순 클론과 차별화 |
| S2 | **무데이터·무광고·완전 무료** | 아동 앱에서 가장 강력한 학부모 신뢰 요소; 구조적 약속(오프라인 아키텍처)으로 번복 불가 |
| S3 | **514개 수작업 큐레이션 단어 자산** | 뜻·발음·예문·카테고리·품사 완비; 경쟁자가 복제하려면 동일한 노력 필요 |
| S4 | **기술 품질 — ESLint 0·TS 0·511 테스트** | 코드 품질이 장기 유지보수성과 App Store 안정성을 보장 |
| S5 | **Sketchy 손그림 디자인 시스템** | 아동 친화적 차별화 UI; 쉽게 모방하기 어려운 통합 디자인 언어 |
| S6 | **완전 오프라인 아키텍처** | 인터넷 없는 환경에서도 작동 → 등·하교 버스, 비행기 등에서 사용 가능 |
| S7 | **명확한 상표 리스크 제거** | 공유 이모지·훅명·문서 정리 완료 → IP 분쟁 리스크 최소화 |

---

## Weaknesses (내부 약점)

| # | 약점 | 설명 |
|---|------|------|
| W1 | **1인 개발 리소스** | 기능 개발 속도 제한; 버그 대응·콘텐츠 확장 동시 진행 어려움 |
| W2 | **브랜드 인지도 제로** | 신규 앱, App Store 리뷰·다운로드 없음 → 초기 신뢰 구축 필요 |
| W3 | **한국어 전용 (초기)** | 글로벌 확장 어려움; 한국 시장에만 집중 → 시장 규모 제한 |
| W4 | **발음 오디오 미구현** | v3.2 예정 기능 — 경쟁 앱 대비 학습 완성도 갭 |
| W5 | **사용자 피드백 루프 없음** | 오프라인 구조상 사용 데이터 수집 불가 → 제품 개선 데이터 부재 |
| W6 | **iOS 전용** | Android 사용자(국내 약 75%) 완전 배제 |
| W7 | **수익 모델 없음** | 지속 가능한 개발 재원 부재 → 장기 콘텐츠 확장 의지 유지 어려울 수 있음 |

---

## Opportunities (외부 기회)

| # | 기회 | 관련성 |
|---|------|--------|
| O1 | **초등 영어 공교육 확대** | 2025년 교육부 초등 3학년 영어 교육 강화 → 어휘 학습 수요 증가 |
| O2 | **아동 앱 개인정보 규제 강화** | COPPA 강화, 국내 아동 개인정보 규제 → 데이터 수집 앱 퇴출 → WordPang 유리 |
| O3 | **학부모의 교육 앱 피로도** | 구독 앱 남용, 광고 노출 불만 증가 → 무료·무광고 포지션 수요 상승 |
| O4 | **유사 게임 방식 학습앱 성공 사례** | Wordle 현상이 단어 게임 수요를 검증 → 교육 변형에 대한 시장 수용성 확인됨 |
| O5 | **교사 채널 미개척** | 초등 영어 교사가 방과 후·자율 학습용 앱 추천 수요 — B2B2C 채널 |
| O6 | **글로벌 K-에듀 관심** | 한국 교육 콘텐츠 해외 인지도 → 영어권·동남아 확장 가능성 |
| O7 | **TTS 기술 성숙** | expo-speech 등 무료 TTS 품질 향상 → 발음 오디오 기능 저비용 구현 가능 |

## Threats (외부 위협)

| # | 위협 | 확률 | 영향 |
|---|------|------|------|
| T1 | **대형 교육 앱 진입** | 중 | 높음 | 듀오링고·링고키즈 등이 유사 퍼즐 모드 추가 시 차별점 희석 |
| T2 | **App Store 정책 변경** | 중 | 높음 | 아동 앱 규제 강화 → 추가 심사 요구, 출시 지연 |
| T3 | **스마트폰 사용 시간 제한 정책** | 높음 | 중 | 교육 당국·학부모의 아동 스마트폰 사용 규제 강화 |
| T4 | **AI 개인 튜터 앱 부상** | 중 | 높음 | ChatGPT 기반 개인화 영어 학습 앱 → 게임 방식 차별화 필요 |
| T5 | **Wordle 클론 포화** | 현재 높음 | 중 | App Store에 Wordle-like 앱 多 → 검색 노출 경쟁 심화 |
| T6 | **Expo SDK 지원 변경** | 낮음 | 높음 | SDK 52 EOL 시 강제 마이그레이션 → 개발 공수 발생 |
| T7 | **1인 개발자 번아웃** | 중 | 높음 | 지속 가능한 유지 불가 시 앱 방치 → 사용자 신뢰 손상 |

| S1(연구 기반) × O1(공교육 확대) | 교육 연구 근거를 교사 채널 진입 도구로 활용 → 학교 추천 |
| S6(오프라인) × O3(학부모 피로도) | "와이파이 없어도 OK, 구독 없어도 OK" 메시지 강화 |
