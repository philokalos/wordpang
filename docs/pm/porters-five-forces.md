# Porter's Five Forces — WordPang

**Date:** 2026-03-22
**Industry:** 한국 초등 영어 교육 앱 (Korean Elementary English Education Apps)
**Framework:** Michael Porter's Five Competitive Forces

---

## 요약 평가

| Force | 강도 | 트렌드 | WordPang 영향 |
|-------|------|--------|-------------|
| 경쟁 우위 (Rivalry) | 🔴 높음 | → 유지 | 차별화 필수 |
| 공급자 파워 (Supplier) | 🟢 낮음 | → 유지 | 유리 |
| 구매자 파워 (Buyer) | 🟡 중간 | ↑ 증가 | 가격 민감도 주의 |
| 대체재 위협 (Substitutes) | 🟡 중간 | ↑ 증가 | 차별화·습관화 필요 |
| 신규 진입 위협 (New Entrants) | 🟡 중간 | ↑ 증가 | 선점 속도 중요 |

**산업 매력도 종합**: **중간** — 차별화된 포지션 (무료·무광고·오프라인)을 유지하면 방어 가능

## Force 1: 경쟁 우위 (Competitive Rivalry)

**강도: 🔴 높음**

### 주요 경쟁자

| 앱 | 모델 | MAU 추정 | 강점 |
|----|------|---------|------|
| 링고키즈 | 구독 ₩9,900/월 | 수십만 | 강력한 캐릭터, 체계적 커리큘럼 |
| 리틀팍스 | 구독 ₩6,900/월 | 수십만 | 영어 스토리 영상, 신뢰도 높음 |
| 듀오링고 | Freemium + 구독 | 수백만 | 브랜드, 게임화, 글로벌 |
| 말해보카 | Freemium + ₩4,900/월 | 수만 | 단어 중심, 한국 맞춤 |
| 무료 유사 앱 | 무료 + 광고 | 다수 | 진입 장벽 없음 |

### 경쟁 강도가 높은 이유

- 한국 교육 앱 시장은 수백 개의 앱이 경쟁하는 레드오션
- 학부모의 앱 교체 주기가 빠름 (3~6개월이면 다른 앱으로 이동)
- 대형사(듀오링고)의 무한한 마케팅 예산
- 무료 앱은 진입 장벽이 낮아 신규 유사 앱이 꾸준히 등장

### WordPang의 포지션

- **차별화**: 완전 무료·무광고·무데이터 + Wordle 방식 추론 게임 (한국 교육 앱에서 유일)
- **위험**: 대형사가 유사 기능을 무료로 출시하면 경쟁력 약화
- **전략적 대응**: Sketchy 디자인 + 한국어 초등 커리큘럼 맥락으로 로컬 차별화 심화

## Force 2: 공급자 파워 (Supplier Power)

**강도: 🟢 낮음**

### 주요 공급자 분석

| 공급자 | 파워 수준 | 이유 |
|--------|---------|------|
| Apple (App Store) | 🔴 높음 | 30% 수수료, 심사 거부 권한, 정책 변경 일방적 |
| Expo / React Native | 🟢 낮음 | 오픈소스, 대체재 많음 (React Native CLI 등) |
| 단어 데이터 | 🟢 낮음 | 자체 큐레이션 (514개), 외부 의존도 없음 |
| TTS/오디오 서비스 | 🟢 낮음 | expo-speech (내장), Google TTS, AWS Polly 등 다수 선택지 |
| Gaegu 폰트 | 🟢 낮음 | 무료 오픈소스 폰트 (Google Fonts) |

### 중요 인사이트

- **Apple이 유일한 고파워 공급자** — App Store 정책 변경, 심사 거부는 통제 불가
- 나머지 기술 스택은 오픈소스 또는 다수 대안 존재 → 락인 위험 낮음
- 단어 데이터를 자체 보유하므로 콘텐츠 공급자 의존도 없음

### 전략적 대응

- Apple Developer Program 규정 준수 철저 (App Store 심사 거부 방지)
- Privacy Policy, COPPA 준수로 앱 삭제 위험 최소화
- 단어 데이터 자체 소유 구조 유지 → 외부 API 불필요

## Force 3: 구매자 파워 (Buyer Power)

**강도: 🟡 중간**

### 구매자 분석

| 구매자 유형 | 파워 수준 | 이유 |
|-----------|---------|------|
| 학부모 (앱 설치 결정자) | 🟡 중간 | 다수 대안 존재, 전환 비용 낮음, 무료 앱은 부담 없이 삭제 |
| 초등학생 (실사용자) | 🟢 낮음 | 학부모 결정권 종속, 직접 구매 없음 |

### 구매자 파워가 중간인 이유

**파워를 높이는 요인**:
- 무료 앱 → 전환 비용 $0, 즉시 삭제 가능
- App Store에 수백 개의 대안 앱 존재
- 학부모는 자녀 반응이 없으면 3~7일 내 삭제

**파워를 낮추는 요인**:
- WordPang의 무료·무광고·무데이터 조합은 학부모 심리적 저항 최소화
- 아이가 사용 중인 앱은 학부모가 쉽게 삭제하지 않음 (아이 의견 존중)
- 통계 화면 → 학부모가 진도 확인 후 지속 허가

### 전략적 대응

- **초기 Aha Moment 설계**: 첫 게임에서 쉬운 단어로 성공 경험 → 아이 만족 → 학부모 허가 연장
- **통계 화면 강화**: 학부모 MOT #2 (유지 결정 시점) 통과 설계
- **배지·Streak**: 아이가 스스로 앱을 찾게 만들어 학부모 삭제 결정을 막음

## Force 4: 대체재 위협 (Threat of Substitutes)

**강도: 🟡 중간**

### 대체재 분석

| 대체재 | 전환 가능성 | 위협 강도 |
|--------|-----------|---------|
| 영어 학원 | 낮음 (비용 대비 비교 불가) | 낮음 |
| 유튜브 영어 채널 | 중간 (무료, 흥미롭지만 수동적) | 중간 |
| 플래시카드 앱 (Quizlet) | 낮음 (게임성 없음, 지루함) | 낮음 |
| 영어 그림책 + 부모 읽어주기 | 낮음 (시간·노력 많이 필요) | 낮음 |
| AI 영어 튜터 (ChatGPT 등) | 중간 (1:1 대화, 하지만 방향 없음) | 중간 → ↑증가 |
| 다른 무료 게임형 앱 | 높음 (직접 대체재) | 높음 |

### 위협이 중간인 이유

- 유사한 **무료 + 게임형 + 교육적** 조합이 많지 않음
- AI 튜터(ChatGPT)의 교육 앱 침투는 증가 추세 → 중기 위협
- 유튜브는 대체재이지만 능동적 학습(단어 추론)을 제공하지 못함

### 전략적 대응

- **습관화 메커니즘 강화**: Streak + 배지 + 복습 루프 → 이탈 비용 증가
- **발음 오디오 추가 (v3.2)**: 학습 완성도 향상 → 유튜브 대비 우위
- **오프라인 가용성**: 유튜브/AI 튜터 대비 확실한 차별점

## Force 5: 신규 진입 위협 (Threat of New Entrants)

**강도: 🟡 중간**

### 진입 장벽 분석

| 장벽 | 수준 | 설명 |
|------|------|------|
| 기술적 장벽 | 낮음 | React Native 생태계 접근성 높음, 개발자 많음 |
| 콘텐츠 장벽 | 중간 | 514단어 큐레이션 + Sketchy 디자인은 모방에 시간 소요 |
| 브랜드 장벽 | 낮음 (초기) | 신규 앱 → 아직 브랜드 미구축 |
| 네트워크 효과 | 낮음 | 싱글 플레이어 앱 → 네트워크 효과 없음 |
| 경쟁사 보복 | 낮음 | 대형 앱들이 신규 소규모 앱에 전략적 대응할 유인 낮음 |
| ASO 경쟁 | 높음 | 키워드 경쟁 심화, 리뷰 수 중요 |

### 위협이 중간인 이유

**위협을 높이는 요인**:
- 앱 개발 비용 하락 (No-code, AI 코딩 도구)
- 한국 교육 앱 시장 매력도 (유입 계속)
- 메타버스/AI 교육 회사들의 시장 진입 가능성

**위협을 낮추는 요인**:
- App Store 리뷰 축적 = 신뢰 장벽 (ReviewGate)
- 간격 반복 데이터 = 사용자 이탈 비용 증가
- "무료·무광고·무데이터" 포지션은 유지 비용 구조로 인해 대형사가 복사하기 어려움

### 전략적 대응

- **빠른 리뷰 축적**: 초기 TestFlight 베타 → 리뷰 50개 목표 → App Store 신뢰도 구축
- **로컬 콘텐츠 심화**: 한국 교과서 연계 단어, 한국어 UI → 글로벌 앱의 진입 장벽
- **선점 속도**: MAU 5,000 달성 전에 강력한 습관 루프 구축

## 전략적 결론

### 가장 중요한 두 가지 힘

1. **경쟁 우위 (Rivalry)**: 가장 큰 위협 — 수백 개의 경쟁 앱
   - **대응**: "무료·무광고·무데이터" + Wordle 방식 = 복사 비용이 높은 차별화 번들

2. **구매자 파워 (Buyer)**: 학부모 전환 비용 = $0
   - **대응**: 초기 Aha Moment + 통계 화면 MOT 설계 → 3일 내 확신 제공

### WordPang에 유리한 힘

- 공급자 파워 낮음 → 비용 구조 유리
- Apple 외 공급자 파워 낮음 → 자율적 기술 의사결정

### 종합 권고

WordPang은 **차별화 전략 (Differentiation Focus)** 을 유지해야 한다:
- 가격 경쟁 (대형사 불가능한 무료) ✅
- 품질 차별화 (Sketchy 디자인, 간격 반복) ✅
- 가치 차별화 (무데이터, COPPA 안전) ✅

**핵심 경계선**: 광고 도입 또는 구독 전환 시 차별화 번들 붕괴 → 대형 경쟁사와 동일한 지형에서 싸우게 됨.
