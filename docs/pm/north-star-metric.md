# North Star Metric — WordPang

**Date:** 2026-03-22

---

## Step 1: Business Game Classification

**WordPang은 Productivity Game을 한다.**

| 비즈니스 게임 | 해당 여부 | 이유 |
|------------|---------|------|
| Attention Game (체류 시간) | ❌ | 체류 시간이 길다고 좋은 게 아님. 빨리 단어를 마스터하는 게 목표 |
| Transaction Game (거래 수) | ❌ | 수익 모델 없음, 거래가 없음 |
| **Productivity Game (목표 달성 효율)** | ✅ | 사용자가 "영어 단어를 효과적으로 배웠다"는 결과를 달성하도록 돕는 것이 핵심 |

**Productivity Game 예시:** Duolingo (언어 습득), Anki (플래시카드 학습), Khan Academy (수학 학습)

WordPang의 비전: *"초등학생이 영어 단어를 외우는 게 아니라 발견하는 세상"*
→ 핵심 가치 전달 = 단어를 장기 기억으로 전환시키는 것

---

## Step 2: North Star Metric

### ✅ 선정: 월간 활성 사용자당 마스터된 단어 수
### **(Words Mastered per Monthly Active User)**

**정의:** 해당 월에 1회 이상 앱을 사용한 사용자 중, `reviewStatus === 'mastered'`(간격 반복 5회 이상 완료)로 전환된 단어 수의 평균

```
NSM = (해당 월 총 새로 mastered된 단어 수) / (월간 활성 사용자 수)
```

### 7가지 기준 검증

| 기준 | 검증 | 설명 |
|------|------|------|
| Easy to Understand | ✅ | "한 달에 사용자 한 명당 몇 단어를 완전히 익혔나?" — 직관적 |
| Customer-Centric | ✅ | 수익·세션·광고 클릭이 아닌, 실제 교육 결과를 반영 |
| Sustainable Value | ✅ | mastered 상태는 장기 기억 전환 — 일시적 플레이가 아닌 실질 학습 |
| Vision Alignment | ✅ | "단어를 발견하고 기억하는 경험" = 비전 그 자체 |
| Quantitative | ✅ | AsyncStorage `wordpang_review_entries`에서 측정 가능 |
| Actionable | ✅ | 게임 완료율, 복습 큐 소화율, 간격 반복 UX 개선으로 직접 영향 |
| Leading Indicator | ✅ | mastered 단어 수 ↑ → 재방문 동기 ↑ → App Store 리뷰 ↑ → 다운로드 ↑ |

### 목표값
| 단계 | 목표 |
|------|------|
| 출시 30일 | > 2 단어/MAU |
| 출시 90일 | > 5 단어/MAU |
| 성숙 단계 | > 10 단어/MAU |

### 고려했으나 탈락한 후보들

| 후보 | 탈락 이유 |
|------|---------|
| 월간 다운로드 수 | 허영 지표 — 설치 후 안 쓰면 의미 없음 |
| 일일 세션 수 | Attention Game 지표 — Productivity Game에 부적합 |
| 평균 streak 일수 | 로그인만 하고 게임 안 해도 streak 증가 가능 |
| 총 학습 단어 수 | 절대값 — MAU 변화 반영 안 됨 (나쁜 사용자 증가해도 수치 오름) |
| D7 리텐션 | 비즈니스 중심적 — 고객 가치 전달 지표가 아님 |

## Step 3: Input Metrics (메트릭 Constellation)

```
                    ┌─────────────────────────────┐
                    │  North Star Metric           │
                    │  Words Mastered / MAU        │
                    │  Target: > 5 (90일)          │
                    └──────────┬──────────────────┘
           ┌───────────────────┼───────────────────────┐
           │                   │                        │
    ┌──────▼──────┐    ┌────────▼──────┐    ┌──────────▼─────┐
    │  Input 1    │    │  Input 2      │    │  Input 3        │
    │ 학습 카드   │    │  복습 큐      │    │  D3 리텐션      │
    │ 완료율      │    │  소화율       │    │                 │
    │ > 70%       │    │  > 60%        │    │  > 30%          │
    └─────────────┘    └───────────────┘    └─────────────────┘
           │                   │                        │
    ┌──────▼──────┐    ┌────────▼──────┐
    │  Input 4    │    │  Input 5      │
    │ 세션 완료율 │    │ 평균          │
    │ (게임 끝까지│    │ streak 일수   │
    │  완료)      │    │               │
    │ > 80%       │    │ > 3일         │
    └─────────────┘    └───────────────┘
```

### Input Metric 1: 학습 카드 완료율
**정의:** 게임 종료 후 학습 카드를 끝까지 확인한 세션 수 / 게임 완료 세션 수

**NSM과의 연결:** 학습 카드를 봐야 단어가 복습 큐에 추가됨 → mastered 전환의 출발점
**목표:** > 70%
**낮을 때 조치:** 학습 카드 UX 개선, 자동 표시 강화

---

### Input Metric 2: 복습 큐 소화율
**정의:** 당일 복습 예정 단어 중 복습 완료된 비율

**NSM과의 연결:** 간격 반복 완료가 곧 mastered 진행. 이 지표가 낮으면 NSM이 올라갈 수 없음
**목표:** > 60%
**낮을 때 조치:** 복습 알림(뱃지) 개선, 복습 화면 진입 동선 단축

---

### Input Metric 3: D3 리텐션율
**정의:** 설치 후 3일 내 재방문한 사용자 비율

**NSM과의 연결:** 재방문해야 복습 큐를 소화할 수 있음. 3일이면 최소 2~3번의 복습 사이클
**목표:** > 30%
**낮을 때 조치:** 온보딩 개선, streak 동기 강화, 첫 배지 빠른 획득 경험

---

### Input Metric 4: 게임 세션 완료율
**정의:** 앱 실행 후 게임을 결과 화면까지 완료한 세션 비율

**NSM과의 연결:** 게임을 완료해야 학습 카드가 나옴 → 중간 이탈 없어야 전체 루프 작동
**목표:** > 80%
**낮을 때 조치:** 게임 난이도 조정 UI 개선, 이탈 지점 분석 (TestFlight 관찰)

---

### Input Metric 5: 평균 streak 일수
**정의:** 활성 사용자의 연속 플레이 일수 평균 (`wordpang_stats.currentStreak`)

**NSM과의 연결:** streak = 매일 재방문 = 간격 반복 사이클 유지. streak 높을수록 mastered 전환 속도 ↑
**목표:** > 3일
**낮을 때 조치:** streak 회복 메커니즘, 연속 기록 배지 알림 강화

## OMTM (One Metric That Matters) — Q1 2026

**D3 리텐션율 (> 30%)**

이유: 출시 초기 최대 과제는 "3일 안에 다시 오게 하는 것". D3 리텐션이 확보되면 복습 큐 소화, 학습 카드 완료, streak 형성이 모두 따라온다. D3가 낮으면 아무리 좋은 간격 반복 알고리즘도 작동하지 않는다.
