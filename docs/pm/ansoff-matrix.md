## Q1: 시장 침투 (Market Penetration)

**현재 제품 × 현재 시장** | 위험도: 🟢 낮음 | 타임라인: 즉시~12개월

### 기회

**기회 1: D3 리텐션 강화 (OST 3대 기회 해결)**
- O1 (첫 게임 후 재방문 동기) → 배지 진행률 표시 (S1-B)
- O3 (복습 가시성 부재) → 홈 화면 복습 배지 (S3-A)
- O4 (Streak 끊김 이탈) → Streak 회복 유예 (S4-A)
- 잠재 효과: D3 리텐션 25% → 30% 달성

**기회 2: App Store 바이럴 루프 강화**
- 이모지 결과 공유 🟩🟧⬜ → 카카오톡 공유 → 친구 다운로드
- 배지 달성 후 "리뷰 남기기" 자연스러운 CTA
- 공유 버튼 prominence 증대 (결과 화면 상단)
- 잠재 효과: K-factor 0.1 → 0.3 (100 다운로드 → 30 추가 유기적 유입)

**기회 3: 맘카페 채널 활성화**
- 개발자 스토리 + "왜 무료인가" 진솔한 글 (광고성 아닌 가치 콘텐츠)
- 맘카페 "영어앱 추천" 스레드에 자연스러운 참여
- 잠재 효과: 오가닉 다운로드 월 200 → 500+ 달성

### 성공 지표
| 지표 | 현재 | 목표 | 기간 |
|------|------|------|------|
| D3 리텐션 | 미측정 | 30% | 90일 |
| 월 다운로드 | 미측정 | 500+ | 60일 |

## Q2: 시장 개발 (Market Development)

**현재 제품 × 신규 시장** | 위험도: 🟡 중간 | 타임라인: 12~24개월

### 기회

**기회 1: 초등 영어 교사 채널 (B2B2C)**
- 타겟: 초등 영어 담당 교사 (~120,000명, 한국)
- 인디스쿨 교사 커뮤니티를 통한 클래스 단위 도입
- WordPang 기존 기능 그대로, 교사용 "과제 지정" UI만 추가
- 조건: MAU 2,000+ 이후, TestFlight 베타로 교사 10명 인터뷰 먼저

**기회 2: 중학생 확장 (한국 시장, 다른 연령층)**
- 현재 초등(4-6글자) → 중등(6-8글자, TOEIC 기초 단어)
- 같은 앱 아키텍처, 다른 단어 데이터셋으로 빠른 확장 가능
- 조건: 초등 Word Mastered per MAU 안정화 후

**기회 3: 해외 시장 — 한국어 학습자 (글로벌)**
- 한류(K-pop, K-drama) 팬의 한국어 학습 수요 급증
- WordPang 방식 + 한국어 단어 = "글로벌 한국어 학습 게임"
- 위험: 영어 초등 시장보다 복잡한 맞춤화 필요 (문자 체계 상이)
- 조건: WordPang 한국 시장 MAU 10,000+ 달성 후 검토

### 우선순위
1. 교사 채널 (MAU 2,000 이후, 빠른 CAC $0)
2. 중학생 확장 (단어 데이터 교체로 저비용)
3. 해외 한국어 (중장기, 별도 프로젝트)

## Q3: 제품 개발 (Product Development)

**신규 제품 × 현재 시장** | 위험도: 🟡 중간 | 타임라인: 12~18개월

### 기회

**기회 1: WordPang Math (수학 개념어 퍼즐)**
- WordPang 무료 앱으로 브랜드 신뢰 구축 → 수학 단어 버전 출시
- 타겟: 초등 수학 어휘 (수, 분수, 도형, 연산 등)
- 가격: $1.99 유료 (WordPang 브랜드 가치 기반)
- 개발 공수: 낮음 — 코어 재사용, 단어 데이터만 교체

**기회 2: WordPang Science (과학 용어 퍼즐)**
- 초등 과학 교과 용어 (동물, 식물, 물질, 지구 등)
- 영문 과학 용어 습득 = STEAM 교육 트렌드 연계
- 가격: $1.99 유료

**기회 3: WordPang 한국어 (초등 국어 어휘)**
- 초등 교과서 국어 어휘 → 한국어로 게임
- 영어 학습 앱과는 다른 세그먼트 (한국어 어휘 부족 아이)
- 가격: $1.99 유료

### WordPang 앱 생태계 모델
```
WordPang (무료) ─── 크로스 프로모션 ───▶ WordPang Math ($1.99)
     │                                   WordPang Science ($1.99)
     └── 브랜드 신뢰 기반                WordPang 한국어 ($1.99)
```

**잠재 수익 (MAU 5,000 기준)**:
- MAU 5,000 × 크로스 전환율 10% × $1.99 = 월 $995

### 검증 실험
- WordPang App Store 설명에 "WordPang Math 출시 예정" 배너 추가
- 클릭률 측정 → 5%+ 클릭 시 개발 진행 결정
- 소요 시간: 출시 후 90일

## Q4: 다각화 (Diversification)

**신규 제품 × 신규 시장** | 위험도: 🔴 높음 | 타임라인: 24개월+

### 기회 (장기 검토)

**기회 1: B2B EdTech 대시보드 플랫폼**
- 학교/학원 전용 단어 학습 관리 플랫폼
- 교사 진도 모니터링 + 학생 단어 과제 + 학부모 리포트
- 요구사항: 서버 개발, B2B 영업, 복잡한 결제 구조
- 위험: 현재 오프라인 아키텍처 전면 변경 + 1인 개발자 부담

**기회 2: 유아 영어 학습 앱 (3~6세)**
- WordPang 방식을 유아용으로 재설계 (음성 인식, 그림 카드)
- 완전히 다른 UX, 다른 SDK (음성 인식 필요)
- 위험: 현재 핵심 역량과 거리가 멈

### 권고: 현재 단계에서 Q4 투자 보류
- Q1 성과 없이 Q4 진입 = 자원 분산으로 Q1도 실패
- MAU 10,000+ 달성 후 Q3 앱 생태계 수익 안정화 → 그 이후 Q4 검토
