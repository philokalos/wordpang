# Team OKRs — WordPang

**팀:** WordPang (1인 개발팀)
**기간:** Q2 2026 (2026-04-01 ~ 2026-06-30)
**회사 목표 연계:** App Store 출시 후 초등 영어 학습 앱 시장에서 유의미한 입지 확보

**전 분기 컨텍스트 (Q1 2026):**
- ✅ v3.1.2 완성 (Sketchy 디자인, IP 정리, 514단어, 511 테스트)

- ⏳ App Store 제출 준비 완료, 출시 미완료
- ❌ 발음 오디오(v3.2) 미완성

## Objective 1: App Store에서 "무료 초등 영어" 카테고리 가시성을 확보한다

*회사 목표 연계: 초등 영어 학습 앱 시장 입지 확보 — 인지도 없이는 어떤 교육 효과도 무의미하다*

| # | Key Result | 기준값 | 목표 | 담당 |
|---|-----------|--------|------|------|
| KR1 | App Store 누적 다운로드 수 | 0 | ≥ 2,000 | 개발자 |
| KR2 | App Store 평균 평점 (리뷰 20개+ 기준) | 0 | ≥ 4.5★ | 개발자 |
| KR3 | 네이버 맘카페·커뮤니티 소개 게시물 노출 | 0 | ≥ 10,000 뷰 (3개 커뮤니티) | 개발자 |
| KR4 | 키워드 "초등영어단어게임" App Store 검색 순위 | 미측정 | Top 30 | 개발자 |

**게임 방지:**
- KR1은 다운로드 수가 아닌 D1 사용자(앱 실제 실행) 기준. 봇 설치 제외.
- KR3는 "좋아요·댓글 있는" 게시물만 카운트 (스팸 방지)

## Objective 2: 사용자가 매일 돌아오는 학습 습관 루프를 완성한다

*회사 목표 연계: North Star(Words Mastered/MAU) 달성의 전제 조건은 재방문이다*

| # | Key Result | 기준값 | 목표 | 담당 |
|---|-----------|--------|------|------|
| KR1 | D7 리텐션율 (App Store Connect) | 미측정 | ≥ 25% | 개발자 |
| KR2 | 평균 streak 일수 (TestFlight 측정 또는 리뷰 분석 추정) | 미측정 | ≥ 3일 | 개발자 |
| KR3 | 학습 카드 완료율 (게임 완료 후 카드까지 확인) | 미측정 | ≥ 70% | 개발자 |
| KR4 | 복습 화면 진입 비율 (앱 실행 중 복습 화면 방문 비율) | 미측정 | ≥ 30% | 개발자 |

**게임 방지:**
- KR2 streak는 "하루 1게임 이상 완료" 기준 (앱 열기만으로 증가하지 않음)
- KR3은 학습 카드를 끝까지 읽은 세션 (0.5초 미만 닫기 제외)

## Objective 3: 제품 학습 완성도를 높여 "배웠다는 느낌"을 완성한다

*회사 목표 연계: 발음 오디오 없이는 영어 단어 학습 앱으로서 경쟁력 미완성*

| # | Key Result | 기준값 | 목표 | 담당 |
|---|-----------|--------|------|------|
| KR1 | 발음 오디오 기능 출시 (v3.2) — TTS 또는 MP3 | 미구현 | 100% 출시 (Q2 말까지) | 개발자 |
| KR2 | 전체 테스트 커버리지 | 76.6% | ≥ 80% | 개발자 |
| KR3 | 접근성 컴포넌트 적용 범위 (VoiceOver 지원) | 9개 컴포넌트 | 전체 게임 화면 컴포넌트 100% | 개발자 |
| KR4 | App Store 리뷰 중 "발음", "소리" 관련 부정 언급 비율 | 미측정 | ≤ 5% | 개발자 |

**게임 방지:**
- KR1은 "출시"가 아닌 "TestFlight 10명 이상이 발음 힌트를 실제 사용한 것이 확인된 출시"
- KR4는 리뷰 20개 이상 모였을 때 측정 시작

## Scoring Guide

| 점수 | 의미 |
|------|------|
| 0.0~0.3 | 심각한 미달 — 원인 분석 필수 |
| 0.4~0.6 | 진전 있으나 미달 |
| **0.7~0.9** | **잘 설정된 stretch 목표 — 이것이 목표** |
| 1.0 | 완전 달성 또는 목표가 너무 낮았음 |
