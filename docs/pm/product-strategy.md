## 1. Vision

> **"초등학생이 영어 단어를 외우는 게 아니라 발견하는 세상을 만든다."**

- **Inspire**: 아이들이 영어를 '어려운 공부'가 아니라 '재미있는 수수께끼'로 경험하도록
- **Aspire**: 교육 연구 기반의 학습 효과 + 게임의 몰입감을 결합하여, 초등 영어 어휘 학습의 표준을 바꾼다
- **Values**: 교육적 정직성(연구 기반), 아동 안전(무광고·무데이터), 접근성(완전 무료·오프라인)

## 3. Relative Costs

**포지션: 고가치·무비용 (Premium Value, Zero Monetary Cost)**

WordPang은 금전적 비용을 0으로 낮추면서도 교육 연구 기반의 고품질 경험을 제공한다. 이는 Southwest Airlines 식 저비용 전략이 아니라, "비용 없는 스타벅스 경험"을 목표로 한다.

| 비용 요소 | WordPang 접근 |
|---------|-------------|
| 서버 인프라 | 없음 (완전 오프라인) → 운영비 $0 |
| 콘텐츠 수집 | 수작업 큐레이션 (514개) → 초기 고정비만 |
| 마케팅 | App Store + 유기적 공유 → 광고비 $0 |
| 개발 유지 | 1인 개발 → 인건비 최소화 |

**경쟁 대비 비용 우위:**
- 링고키즈 월 ₩9,900, 듀오링고 연 ₩99,000 vs. WordPang 무료
- 광고 없음 → 아동 안전성 프리미엄

## 4. Value Proposition

### 세그먼트 1 (초등학생) 가치 제안

| 요소 | 내용 |
|------|------|
| **What Before** | 단어장 암기는 지루하고, 단어를 외워도 금방 잊어버린다 |
| **How** | 색깔 힌트로 단어를 직접 추리 → 교정 피드백(+494% 기억력) → 간격 반복 자동 스케줄 |
| **What After** | 게임을 하는 것 같은데 단어가 저절로 외워진다; 학원 시험에서 아는 단어가 늘어난다 |
| **Alternatives** | 교과서 단어장, 단어 암기 앱, 듀오링고 |

### 세그먼트 2 (학부모) 가치 제안

| 요소 | 내용 |
|------|------|
| **What Before** | 아이에게 줄 만한 영어 앱이 없다 — 광고가 있거나, 구독료가 부담되거나, 개인정보 수집이 걱정된다 |
| **How** | 무료 + 무광고 + 무데이터 수집 + 4+ 연령등급 + 통계 화면으로 학습 진도 확인 |
| **What After** | 걱정 없이 아이에게 맡길 수 있고, 통계로 성장을 확인할 수 있다 |
| **Alternatives** | 링고키즈, 뽀로로 영어, 교과서 부록 |

## 5. Trade-offs (우리가 하지 않을 것들)

| 하지 않을 것 | 이유 |
|------------|------|
| 서버 연동 / 클라우드 동기화 | 개인정보 수집 없는 정책의 핵심 — 학부모 신뢰의 원천 |
| 광고 탑재 | 아동 앱의 경험 품질과 학부모 신뢰를 파괴 |
| 구독 모델 | 접근성(무료) = 차별점 핵심 — 구독으로 전환 시 경쟁자와 동질화 |
| Android 출시 (단기) | iOS 단일 플랫폼 집중으로 품질 유지 |
| 멀티플레이어 / 랭킹 | 아동 소셜 기능 = COPPA 리스크 + 개발 복잡도 폭증 |
| 음성인식 | 마이크 권한 = 학부모 거부감, 아동 개인정보 |
| 영어 전반 학습 | 어휘 특화 = 차별화 포지션 유지 |

## 6. Key Metrics

### North Star Metric
> **"월간 활성 학습자당 마스터된 단어 수 (Words Mastered per MAU)"**

이 지표가 올라가면 = 더 많은 아이들이 더 많은 단어를 장기 기억으로 전환하고 있다는 의미. 제품의 교육적 목적과 비즈니스 목적을 동시에 측정.

### OMTM (One Metric That Matters, Q1 2026)
> **"첫 3일 완주율 (3-Day Retention)"**

출시 초기 핵심 과제: 첫 3일 이내 재방문. 이 지표가 30%+ 이면 습관 형성 루프가 작동 중.

### Supporting Metrics

| 메트릭 | 목표 |
|--------|------|
| D1 Retention | > 40% |
| D7 Retention | > 25% |
| 평균 streak 일수 | > 3일 |
| 세션 완료율 (학습 카드까지) | > 70% |
| 복습 큐 소화율 | > 60% |
| App Store 평점 | 4.5★+ |

## 7. Growth

### Growth Model: Product-Led Growth (PLG)

서버 없음 → 전통적 SaaS PLG 루프 불가. 대신 **Content × Community PLG**:

```
아이가 앱 발견 (App Store 검색)
→ 무료·무광고·무데이터로 즉시 설치
→ 첫 게임 10분 내 첫 승리 경험
→ 공유 기능 (이모지 결과 🟩🟧⬜)
→ 친구/부모에게 공유 → 바이럴
→ App Store 리뷰 → 노출 증가
```

### 주요 채널

| 채널 | 설명 | 우선순위 |
|------|------|---------|
| App Store 검색 | "초등영어", "영어단어게임" 키워드 | P0 |
| 학부모 커뮤니티 | 네이버 카페, 맘카페, 육아 블로그 | P0 |
| 결과 공유 | 이모지 그리드 SNS 공유 | P1 |
| 교사 추천 | 초등 영어 선생님 → 반 전체 | P2 |
| YouTube | 사용법 영상, 교육 채널 협업 | P2 |

### Unit Economics

- CAC: $0 (유기적 성장, 광고비 없음)
- 수익: 현재 $0 (무료) → 향후 교사 버전 B2B 가능
- LTV: 장기 무료 사용 → 브랜드 신뢰 → 후속 유료 제품 기반

## 9. Defensibility (Can't / Won't Copy)

### 모방하기 어려운 이유

| 방어 요소 | 설명 |
|----------|------|
| **콘텐츠 자산** | 514개 단어 × 뜻·발음·예문 수작업 큐레이션 → 복제 시 동일한 노력 필요 |
| **교육 연구 설계** | 4가지 교육 이론 통합 설계 → 단순 클론으로는 효과 재현 불가 |
| **무데이터 정책** | 학부모 신뢰 = 구조적 약속. 서버 연동 경쟁자는 이 포지션 불가 |
| **오프라인 아키텍처** | 복잡한 오프라인 간격 반복 + 업적 시스템 → 설계 복잡도 진입 장벽 |
| **App Store 리뷰 자산** | 출시 후 누적 리뷰 → 신규 진입자 대비 신뢰도 우위 |

### 핵심 가설 (전략이 성립하기 위해 참이어야 하는 것들)

| 가설 | 검증 방법 |
|------|---------|
| H1: 아동은 퍼즐 방식으로 단어를 배우는 것을 암기보다 선호한다 | App Store 리뷰, D7 리텐션 측정 |
| H2: 학부모는 무료·무광고를 유료 기능보다 더 높이 평가한다 | 리뷰 키워드 분석 ("광고", "무료") |
| H3: App Store 유기적 검색만으로 충분한 다운로드가 가능하다 | 출시 후 60일 다운로드 추적 |
| H4: 간격 반복 알고리즘이 실제 단어 기억 향상으로 이어진다 | learned_words mastered 전환율 |
