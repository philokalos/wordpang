# Product Name Ideas — WordPang 생태계 확장

**Date:** 2026-03-22
**맥락:** WordPang 브랜드 확장 시 형제 앱 네이밍 + 현재 앱 이름 검토
**브랜드 가치:** 재미, 발견, 무료, 초등 친화, 스케치 스타일

---

## Part 1: 현재 앱 이름 평가 — "WordPang"

| 기준 | 평가 | 점수 |
|------|------|:---:|
| 발음 용이성 | 한국어/영어 모두 쉬움 ("워드팡") | ✅ |
| 기억하기 쉬움 | "Word" + "Pang" 조합 — 독특함 | ✅ |
| 브랜드 적합성 | 어린이 느낌, 가벼움, 재미 | ✅ |
| 도메인/상표 | App Store 등록 완료, Wordle 상표 충돌 해결 완료 | ✅ |
| 검색 발견성 | "WordPang" 고유 단어 → ASO 키워드 경쟁 낮음 | ✅ |

**결론: WordPang은 유지. 형제 앱 네이밍에 집중.**

---

## Part 2: 형제 앱 5가지 이름 제안

### Name 1: **WordPang Math**

**이름:** WordPang Math

**근거:**
- WordPang 브랜드 패밀리 확장의 가장 직관적인 패턴 (듀오링고 → 듀오링고 Math와 유사)
- "Math" = 명확한 카테고리, 학부모 즉시 이해
- WordPang 다운로더가 "WordPang Math"를 App Store 검색 → 자연스러운 크로스셀

**브랜드 적합성:**
- 동일 Sketchy 디자인 시스템 사용 가능
- "Math" 단어 추론 게임 = 수학 용어를 Wordle 방식으로 → 개념 즉시 전달

**기억하기 쉬움:**
- WordPang 사용자라면 "아, 수학 버전이구나" 즉시 이해
- 단순 명료, 오해 없음

**도메인/상표:**
- "WordPang" 상표 이미 보유 → 패밀리 브랜드로 확장 가능
- wordpangmath.com 등록 검토

---

### Name 2: **PuzzlePang**

**이름:** PuzzlePang

**근거:**
- "Word"에서 "Puzzle"로 확장 — 영어 단어에 국한되지 않는 상위 브랜드
- 수학, 과학, 국어 등 다양한 교과목 포함 가능한 상위 카테고리
- "-Pang" 접미사 유지로 브랜드 연속성

**브랜드 적합성:**
- "퍼즐팡" — 한국 어린이들에게 친숙한 느낌
- WordPang을 "PuzzlePang English"로 리브랜딩 가능 (선택적)

**기억하기 쉬움:**
- "Puzzle" + "Pang" = 명확한 연상, 어린이 게임 느낌
- 부모와 아이 모두 이해 가능

**도메인/상표:**
- puzzlepang.com 검토 필요
- WordPang과 별개 상표 등록 필요

---

### Name 3: **NumberPang**

**이름:** NumberPang (수학 앱용)

**근거:**
- WordPang의 자매 앱에 "Number"로 명확하게 수학 분야 표시
- "-Pang" 패밀리 연속성 + 어린이들에게 "숫자" 연상 직관적
- "넘버팡" — 한국에서 발음하기 쉽고 귀여운 느낌

**브랜드 적합성:**
- WordPang (단어) + NumberPang (숫자) = 완벽한 대칭 브랜드 구조
- 같은 Sketchy 디자인, 같은 게임 메커니즘

**기억하기 쉬움:**
- WordPang 알면 NumberPang 바로 이해
- 학부모 카톡방: "WordPang 있어요? NumberPang도 있어요!"

**도메인/상표:**
- numberpang.com 검토 필요

---

### Name 4: **PangWorld** (브랜드 플랫폼)

**이름:** PangWorld

**근거:**
- "Pang" 시리즈를 포괄하는 브랜드 플랫폼 이름
- WordPang, NumberPang, SciencePang 등의 상위 우산 브랜드
- "월드" = 어린이들이 좋아하는 세계 확장 개념 (Pokémon World 연상)

**브랜드 적합성:**
- 앱 생태계(App Ecosystem) 전략 실행 시 플랫폼 브랜드로 활용
- 각 앱은 "PangWorld에서" 나온 것으로 크로스 프로모션

**기억하기 쉬움:**
- 개별 앱 이름보다 기억하기 쉬운 브랜드 우산
- "팡월드" — 어린이 게임 세계 느낌

**도메인/상표:**
- pangworld.com / pangworld.io 검토 필요
- 장기 브랜드 자산으로 투자 가치 있음

---

### Name 5: **SpellPang** (발음/철자 특화 앱용)

**이름:** SpellPang

**근거:**
- v3.2 발음 오디오 기능 추가 후 "발음+철자" 특화 앱으로 분리 가능
- "Spell" = 철자 쓰기 + 발음 → 영어 기초 학습의 핵심
- "-Pang" 패밀리 연속성

**브랜드 적합성:**
- WordPang (추론 게임) vs. SpellPang (철자/발음 연습) = 명확한 역할 분리
- 초등 1~2학년 (알파벳 시작) 타겟으로 더 낮은 연령층 커버

**기억하기 쉬움:**
- "스펠팡" — "맞춤법"의 영어 "Spell"을 자연스럽게 연상
- 학부모: "발음 연습하는 앱이구나"

**도메인/상표:**
- spellpang.com 검토 필요
- WordPang 상표 이후 패밀리 등록 가능

## 추천 네이밍 전략

| 단계 | 앱 이름 | 타이밍 |
|------|---------|--------|
| 지금 | WordPang (유지) | 현재 |
| Phase 2 (MAU 2,000+) | NumberPang (수학 첫 형제 앱) | 검증 후 |
| Phase 3 (MAU 5,000+) | WordPang Science / SpellPang | 생태계 확장 |
| 장기 | PangWorld (브랜드 플랫폼) | MAU 10,000+ |

### 핵심 네이밍 원칙

1. **"-Pang" 접미사 유지** — 브랜드 패밀리 연속성의 핵심 자산
2. **교과목명 선두** — Word / Number / Science / Spell로 즉시 카테고리 전달
3. **한국어 발음 친화성 확인** — "팡" 접미사는 이미 검증됨
4. **App Store 크로스 링크 준비** — 각 앱에서 형제 앱 소개
