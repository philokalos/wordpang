# Marketing Ideas — WordPang

**Date:** 2026-03-22
**타겟:** 초등 3~4학년 학부모 (비치헤드 세그먼트)
**예산:** $0 (완전 오가닉)
**제약:** 개인 개발자, 광고 집행 불가, 서버 없음

---

## Idea 1: 개발자 진솔한 스토리 — 맘카페 진입

**채널:** 네이버 맘카페 / 다음 카페 / 카카오 육아 커뮤니티

**핵심 메시지:**
> "아이한테 '영어 공부해' 라고 말하기 싫어서 만들었어요. 광고도 없고, 데이터도 안 모으고, 구독료도 없어요. 그냥 아이가 스스로 켜면 좋겠어서요."

**왜 효과적인가:**
- 맘카페에서 "광고성 홍보글"은 즉시 삭제되지만, 개발자의 진솔한 스토리는 공감과 바이럴 유발
- 학부모는 "왜 무료인가?"에 항상 의심 → 개발자가 직접 이유를 밝히면 신뢰 급상승
- "내 아이를 위해 만들었다"는 메시지 = 학부모 감정 코드와 완벽 일치
- 링고키즈/리틀팍스의 기업 마케팅과 대비되는 인간적 이야기

**비용 효율:**
- 비용: $0 (시간 2~3시간)
- 잠재 도달: 맘카페 구독자 수만 명 (공감 받으면 베스트글 등재)
- 기대 다운로드: 1게시물 당 100~500건 추정

**실행 방법:**
1. "초등 영어" 또는 "영어 앱 추천" 인기 게시판 파악
2. 300자 내외 개발 배경 스토리 + 스크린샷 2~3장
3. 댓글 질문에 개발자 직접 성실하게 답변
4. 절대 하지 말 것: "무료 앱 소개합니다" 광고성 제목

## Idea 2: 학교 학부모 카톡방 바이럴 키트

**채널:** 카카오톡 단체방 (학교 반 학부모 그룹)

**핵심 메시지:**
> "🟩🟧⬜ 우리 아이 오늘 게임 결과예요! WordPang이라는 앱인데 광고도 없고 완전 무료예요. 얘가 혼자서 켰어요 😂"

**왜 효과적인가:**
- 학교 반 학부모 카톡방 = 한국에서 가장 신뢰도 높은 추천 채널
- 이모지 결과 그리드가 호기심 유발 → "이게 뭐야?" → 앱 이름 공유 → 즉시 설치
- "혼자서 켰다" = 학부모가 가장 원하는 결과를 공유 → 동질감 + 설치 욕구
- 학교 단위 전파 = 클래스 내 친구들이 같이 하면 사회적 강화

**비용 효율:**
- 비용: $0
- 전파 구조: 1개 학교 반(30가정) → 5개 반 → 150가정 잠재 도달
- App Store 리뷰에 "친구 추천으로 알았어요" 유도 가능

**실행 방법:**
1. 결과 화면 공유 버튼에 "카카오톡으로 공유" 옵션 구현
2. 공유 텍스트 기본값: "오늘 WordPang 결과예요! 🟩🟧⬜ [다운로드 링크]"
3. 배지 달성 시 "친구에게 자랑하기" CTA 추가 → 자연스러운 공유 유도

## Idea 3: App Store 최적화 (ASO) — 검색 의도 타겟

**채널:** App Store 검색 결과 (한국 App Store)

**핵심 메시지 (App Store 제목/서브타이틀):**
> "WordPang — 초등 영어 단어 게임"
> 서브타이틀: "광고 없는 무료 영어 게임"

**왜 효과적인가:**
- 학부모의 App Store 검색 키워드: "초등영어앱", "어린이영어게임", "영어단어게임무료"
- "광고 없는 무료"를 제목에 포함 → 검색 노출 + 클릭률 동시 향상
- 스크린샷 전략: 1번(게임 장면) → 2번(배지 화면) → 3번(통계 화면) → 4번(무광고·무데이터 강조)
- 경쟁 앱이 "광고 없음"을 전면에 내세우지 않음 → 차별화 포지션

**비용 효율:**
- 비용: $0 (시간 4~6시간 초기 설정)
- 검색 발견성: 올바른 키워드 최적화 시 월 다운로드 200~500건 추가 가능
- 지속 효과: 한 번 최적화 후 지속 오가닉 유입

**실행 방법:**
1. App Store Connect 키워드 100자 최적화 — "초등영어,단어게임,영어공부,어린이영어,무료영어"
2. App Preview 동영상 추가: 게임 플레이 15초 + 배지 달성 순간
3. 리뷰 응답: 모든 리뷰(긍정/부정)에 24시간 내 개발자 답변

## Idea 4: 초등 영어 교사 인디스쿨 공략

**채널:** 인디스쿨 (초등교사 커뮤니티) + 교사 유튜브/블로그

**핵심 메시지:**
> "수업 시작 전 5분 — WordPang 한 게임으로 단어 분위기 잡기. 광고 없고, 데이터 안 모아서 학부모 연락 올 일 없어요 😊"

**왜 효과적인가:**
- 인디스쿨 게시물 1개 = 교사 수만 명 도달 가능
- 교사 추천 = 학부모 신뢰 즉시 최고치 ("선생님이 추천해줬어요")
- 클래스 20~30명 단위 전파 → 학부모 카톡방 전파와 시너지
- 교사가 "무광고·무데이터" 포인트에 안도 → 수업 중 사용 허가 가능

**비용 효율:**
- 비용: $0
- 잠재 도달: 인디스쿨 회원 30만+ 교사
- 기대 전환: 교사 1명 → 학생 25명 → 학부모 25명 잠재 전파

**실행 방법:**
1. 인디스쿨 "영어 수업 도구" 게시판에 교사용 활용법 가이드 게시
2. 수업 활용 예시: "오늘의 단어 맞히기 게임 — 누가 먼저 맞힐까?" 경쟁 요소
3. 교사 피드백 직접 반영 약속 → 인디스쿨 팔로우 커뮤니티 형성

## Idea 5: "아이가 혼자 켰어요" 순간 캡처 캠페인

**채널:** 인스타그램 / 카카오스토리 + App Store 리뷰

**핵심 메시지:**
> "아이가 '엄마, WordPang 해도 돼요?' 하는 순간을 기다리고 있어요."
> 해시태그: #아이가혼자켰어요 #WordPang #무광고영어앱

**왜 효과적인가:**
- Product Vision("아이가 먼저 하고 싶다고 말하는 날")을 마케팅 언어로 직접 전환
- 학부모의 가장 원하는 결과 = "강요 없이 자발적 학습" → 감정적 공명
- UGC(사용자 생성 콘텐츠) 유도 → 학부모가 자녀 게임 화면 스크린샷 공유
- App Store 리뷰에 "아이가 혼자 켜요"라는 실제 경험 자연스러운 유도

**비용 효율:**
- 비용: $0
- 해시태그 활성화 시 자생적 콘텐츠 생성 → 브랜드 인지도 구축
- App Store 리뷰 품질 향상 → 검색 노출 + 전환율 개선

**실행 방법:**
1. 배지 달성 화면에 "#아이가혼자켰어요 #WordPang" 공유 버튼 추가
2. App Store 스크린샷 마지막: "아이가 먼저 '게임 하고 싶어요'라고 말하는 날" 비전 텍스트
3. 인스타그램 개발자 계정 개설 → 첫 3개월간 꾸준히 개발 일지 + 아이 반응 스토리 게시
