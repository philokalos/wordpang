# Competitive Analysis — WordPang

**Date:** 2026-03-22
**분석 범위:** 초등학생 영어 단어 학습 앱 (한국 iOS 시장)
**분석 목적:** 제품 포지셔닝 확정, App Store 메시지 전략, 로드맵 우선순위

---

## Market Overview

한국 초등 영어 학습 앱 시장은 구독 기반 프리미엄 앱(링고키즈, 말해보카)과 무료·광고 기반 앱(듀오링고)으로 양분되어 있다. 아동 개인정보 규제 강화(국내 아동 개인정보법, 글로벌 COPPA 강화)와 학부모의 구독 피로도가 동시에 심화되는 시점이다. "완전 무료 + 무광고 + 무데이터 수집 + 교육 연구 기반" 포지션은 현재 시장에 공백으로 존재한다.

## 경쟁사 분석

### 직접 경쟁자 (Direct Competitors)

#### 1. 듀오링고 (Duolingo)
| 항목 | 내용 |
|------|------|
| 타겟 | 영어 학습자 전반 (성인 중심, 아동 포함) |
| 포지셔닝 | "세계 1위 언어 학습 앱 — 매일 5분으로 새 언어 배우기" |
| 가격 | 무료 (광고) / Super Duolingo $6.99/월 |
| 강점 | 거대 브랜드 인지도, 게이미피케이션 선구자, 45개 언어, streak 시스템 |
| 약점 | 영어 전반 (어휘 특화 아님), 광고 있음(무료), 개인 데이터 수집, 초등 어휘 커리큘럼 없음 |
| App Store 평점 | 4.7★ |
| 최근 동향 | Duolingo Max (GPT-4 기반 대화) 추가 — 어휘보다 회화 강화 방향 |

#### 2. 링고키즈 (LingoKids)
| 항목 | 내용 |
|------|------|
| 타겟 | 2~8세 영어 학습 (유아·저학년 중심) |
| 포지셔닝 | "아이들을 위한 놀이 기반 영어 학습" |
| 가격 | 무료 체험 / ₩9,900/월 구독 |
| 강점 | 캐릭터·스토리 기반 몰입, 유아 특화 UI, 오프라인 일부 지원 |
| 약점 | 유아 편중 (초등 3학년+ 급격히 시시해짐), 구독 비용 부담, 개인정보 수집 |
| App Store 평점 | 4.6★ |
| 최근 동향 | 초등 고학년 콘텐츠 확장 시도 중 — 잠재 경쟁 심화 가능 |

#### 3. 말해보카

## Differentiation Opportunities

| # | 기회 | 방어 가능성 | 이유 |
|---|------|-----------|------|
| D1 | **"무데이터 + 교육 효과" 포지션 독점** | 높음 | 오프라인 아키텍처는 단순 설정이 아닌 구조적 약속 — 경쟁사는 데이터 수집으로 수익화하므로 전환 불가 |
| D2 | **간격 반복 + 게임 메커니즘 결합** | 중간 | 두 요소를 동시에 구현한 아동 특화 앱이 현재 없음. 복제는 가능하나 시간 소요 |
| D3 | **초등학생 맞춤 514개 큐레이션 단어** | 중간 | 경쟁사들은 일반 사전 기반 — WordPang은 교과과정·카테고리 맞춤 큐레이션 자산 |
| D4 | **교사 채널 (B2B2C) 선점** | 낮음~중간 | 현재 경쟁사들이 교사 추천 채널을 적극 활용하지 않음 — 선점 가능 |

## Competitive Threats

| # | 위협 | 확률 | 대응 |
|---|------|------|------|
| T1 | **링고키즈의 초등 고학년 확장** | 높음 | 교육 연구 기반·간격 반복을 전면에 내세워 "더 효과적인 학습" 포지션 강화 |
| T2 | **듀오링고의 어휘 특화 모드 추가** | 중간 | 무데이터·무광고 포지션은 듀오링고가 절대 따라올 수 없음 — 이 메시지 강화 |
| T3 | **아동 특화 AI 튜터 앱 부상** | 낮음~중간 | 게임 방식의 반복 구조는 AI 대화와 보완적 — "함께 쓰는 앱"으로 리포지셔닝 가능 |
| T4 | **Wordle 클론 앱 포화** | 현재 높음 | 교육 연구 기반·초등 커리큘럼·손그림 UI로 일반 클론과 확실히 차별화 |

## Recommendations

### Double Down On (강점 극대화)
- **"무료·무광고·무데이터 수집"** — 학부모 대상 마케팅의 첫 번째 메시지. App Store 설명 첫 줄에 배치.
- **교육 연구 기반 스토리텔링** — Butler et al. (+494%), Nakata 2015 (+53%) 수치를 학부모 채널에서 적극 활용
- **간격 반복 자동화** — 경쟁사 대비 유일한 자동 복습 스케줄 — 사용 설명에서 강조

### Close the Gap On (격차 해소)
- **발음 오디오 (v3.2 P0)** — 말해보카, 링고키즈, 듀오링고 모두 있음. 미구현이 학습 완성도 갭으로 인식됨
- **App Store ASO 최적화** — 키워드, 스크린샷 헤드라인, 리뷰 요청 전략 필요
- **온보딩 첫 승리 경험 30초 내** — 듀오링고는 가입 없이 즉시 첫 레슨 시작

### Ignore (대응 불필요)
- 링고키즈·리틀팍스의 캐릭터 IP 경쟁 — 독립 개발자가 따라갈 수 없고 WordPang의 차별점도 아님
- 듀오링고의 GPT-4 기반 회화 기능 — WordPang의 어휘 퍼즐 포지션과 다른 Job-to-be-Done
