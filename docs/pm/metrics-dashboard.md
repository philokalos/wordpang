# Product Metrics Dashboard — WordPang

**Date:** 2026-03-22
**특이사항:** WordPang은 완전 오프라인 앱으로, 서버 사이드 분석 도구(Amplitude, Mixpanel 등) 사용 불가. 모든 지표는 App Store Connect 데이터 + 로컬 AsyncStorage 기반 추산으로 측정.

---

## 1. North Star Metric

> **월간 활성 사용자당 마스터된 단어 수 (Words Mastered per MAU)**
>
> 정의: 해당 월에 1회 이상 플레이한 사용자 중, `reviewStatus === 'mastered'`로 전환된 단어 수의 평균
>
> 왜 이 지표인가: 다운로드 수(허영 지표)나 세션 수(활동 지표)와 달리, 이 지표가 올라가면 실제 교육 목적(장기 기억 전환)이 달성되고 있다는 의미. 비즈니스 성공(재방문·리뷰·추천)의 선행 지표.

| 항목 | 값 |
|------|-----|
| 측정 방법 | App Store Connect MAU × 앱 내 mastered 평균 (설문 또는 TestFlight 측정) |
| 목표 | > 5단어/MAU (출시 90일 내) |
| 경보 임계값 | < 2단어/MAU 시 핵심 루프 검토 |
| 검토 주기 | 월간 |

## 7. Alert Rules

| 경보 | 임계값 | 채널 | 대응 |
|------|--------|------|------|
| 크래시 급증 | 크래시율 > 1% | App Store Connect 이메일 | 즉시 핫픽스 검토 |
| 평점 하락 | 평균 평점 < 4.0★ | 수동 모니터링 | 리뷰 내용 분석 후 패치 |
| D3 리텐션 < 15% | 출시 첫 2주 | App Store Connect | 온보딩 플로우 재검토 |
| 빌드 실패 | EAS Build 실패 | EAS 이메일 | 즉시 오류 수정 |

## 8. 측정 도구 (오프라인 앱 특화)

| 도구 | 용도 | 비고 |
|------|------|------|
| **App Store Connect** | 다운로드, 리텐션, 크래시, 평점 | 무료, Apple 기본 제공 |
| **TestFlight + 수동 관찰** | Activation·완료율 측정 | 5~10명 베타 테스터로 측정 |
| **AsyncStorage 직접 덤프** | streak·mastered 단어 분포 분석 | 개발자 직접 데이터 추출 |
| **App Store 리뷰 텍스트 분석** | 정성 피드백, 기능 요청 | 수동 분류 |

> **왜 Amplitude/Mixpanel 사용 안 하는가:**
> WordPang은 개인정보 무수집 정책의 구조적 약속으로 모든 분석 SDK를 배제한다. 이는 단순 설정이 아닌 아키텍처적 약속이며, COPPA·PIPA 완전 준수의 핵심이다.

## 9. Vanity Metrics 주의사항

| 허영 지표 | 왜 조심해야 하는가 | 대신 볼 지표 |
|----------|-----------------|------------|
| 총 다운로드 수 | 설치만 하고 안 쓰는 사용자 포함 | D3/D7 리텐션 |
| 총 플레이 횟수 | 같은 사용자의 반복 플레이 포함 | 학습 카드 완료율 |
| 배지 해금 수 | 게임 실력이지 학습 효과가 아님 | mastered 단어 수 |
| App Store 노출 수 | 전환율 모르면 의미 없음 | Impressions → CVR |
