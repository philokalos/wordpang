# Feature Prioritization — WordPang v3.2

**Date:** 2026-03-22
**Framework:** ICE Score (Impact × Confidence × Ease) + 전략 정렬도
**목표:** D3 리텐션 25% → 30% (출시 후 90일)

---

## 평가 방법론

**ICE Score** = Impact × Confidence × Ease (각 1~10 점수)
- **Impact**: D3 리텐션 또는 NSM(Words Mastered per MAU)에 미치는 영향
- **Confidence**: 효과 발생 확신도 (리서치/데이터 기반)
- **Ease**: 개발 용이성 (1 = 매우 어렵, 10 = 매우 쉬움)

**전략 정렬도**: OST 우선순위(O1/O3/O4) + PRD P0/P1 기준과의 일치도

## 상위 5개 기능 — 최종 추천

### 🥇 1순위: 홈 화면 복습 배지 (F2) | ICE: 512

**근거:**
- 복습 큐 데이터가 이미 존재 → UI 추가만 필요 (Ease: 8)
- OST O3 직접 해결 — "복습할 단어가 있는지 알 방법 없다" (Opportunity Score 0.72)
- D3 리텐션에 직접 기여: 앱 재방문 트리거를 홈 화면에서 제공
- SketchyBox 컴포넌트 재사용으로 빠른 구현 가능

**핵심 가설:** 홈 화면에 복습 N개 카드가 보이면 복습 화면 진입률이 현재 추정 10% → 30%로 증가

**구현 범위:** 홈 화면 상단 카드 컴포넌트 추가 (3시간 예상)

---

### 🥈 2순위: Streak 회복 유예 (F1) | ICE: 504

**근거:**
- OST O4 직접 해결 — "streak 끊기면 다시 시작하기 싫어진다" (Opportunity Score 0.68)
- Churn Trigger #1 (Customer Journey Map) 직접 대응
- Duolingo의 streak freeze와 유사하지만 더 단순 (구매 없이 1일 자동)
- 심리적 효과 높음: 실패 → 복구 가능 인식 전환

**핵심 가설:** streak 끊긴 다음 날 "오늘 복구 가능!" 메시지가 D1 재방문율을 10% → 35%로 높인다

**구현 범위:** storage.ts streak 로직 수정 + 홈 화면 메시지 조건부 표시 (4시간 예상)

---

### 🥉 3순위: 배지 진행률 표시 (F3) | ICE: 441

**근거:**
- OST O1 직접 해결 — "첫 게임 후 또 하고 싶다는 느낌 안 든다" (Opportunity Score 0.76, 최우선)
- 결과 모달 하단에 "다음 배지까지 2게임 남았어요!" → 도파민 루프
- Ease 9: achievements.ts 데이터 이미 있음, 진행바 UI만 추가
- A/B 테스트 가능 (OST Experiment 1)

**핵심 가설:** 배지 진행률을 보여주면 결과 화면에서 "다시 하기" 탭률이 30% → 50%로 증가

**구현 범위:** 결과 모달 배지 진행바 섹션 추가 (2시간 예상)

---

### 4순위: Streak 끊김 위로 화면 (F10) | ICE: 324

**근거:**
- F1 (Streak 회복)과 시너지: 회복 메시지 + 위로 UX를 함께 구현
- "괜찮아, 오늘부터 다시 시작하면 돼! 🔥" → 심리적 장벽 제거
- Ease 9: 결과 모달 조건부 메시지 추가
- F1과 묶어서 개발하면 추가 비용 최소 (공유 스프린트)

**구현 범위:** 결과 모달 streak 0 상태 UI 분기 처리 (1~2시간 예상)

---

### 5순위: 발음 오디오 — 학습 카드 (F4) | ICE: 252

**근거:**
- Customer Journey Map MOT #2 (학부모 지속 허가)에 기여 — "아이가 발음도 배운다" 확인
- PRD P0 기능으로 v3.2 핵심 요소
- Ease 6: expo-speech 또는 오디오 파일 접근 방식 결정 필요
- D3 리텐션보다는 교육 가치 향상 (NSM 기여)

**구현 범위:** expo-speech TTS 통합 + 스피커 버튼 UI + 514개 단어 커버 (6~8시간 예상)
