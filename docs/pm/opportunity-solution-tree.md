## Level 3: Solutions

### O1: "첫 게임이 끝났는데 또 하고 싶다는 느낌이 안 든다"

**근본 원인:** 첫 게임이 끝나는 순간이 에너지가 떨어지는 구조. 다음 액션이 명확하지 않음.

| 솔루션 | 관점 | 설명 |
|--------|------|------|
| S1-A | 🎮 PM | **"연속 게임" 버튼 추가** — 결과 화면에 "다시 도전" 버튼을 "공유" 버튼보다 크게. 다음 게임 즉시 시작. |
| S1-B | 🎨 Designer | **배지 진행률 표시** — 결과 화면에 "다음 배지까지 2게임 남았어요!" 진행바 표시 → 도파민 루프 |
| S1-C | ⚙️ Engineer | **다음 단어 미리보기** — 결과 모달에서 "다음 단어 카테고리: 🐾 동물" 힌트 표시 → 호기심 유발 |
| S1-D | 🎮 PM | **일일 목표 설정** — 온보딩에서 "오늘 몇 단어 도전할까요? (1 / 3 / 5)" 선택 → 진행률 표시 |

**가장 유망한 솔루션: S1-B** (개발 공수 낮음, 배지 시스템과 시너지)

---

### O3: "복습할 단어가 있다는 걸 알 방법이 없다"

**근본 원인:** 복습 큐가 있어도 홈 화면에서 가시성이 낮음. 복습 탭은 메뉴 안에 숨겨짐.

| 솔루션 | 관점 | 설명 |
|--------|------|------|
| S3-A | 🎮 PM | **홈 화면 복습 배지** — "복습 단어 7개" 배지(뱃지)를 홈 화면 상단에 카드 형태로 표시 |
| S3-B | 🎨 Designer | **앱 아이콘 뱃지** — iOS 앱 아이콘에 복습 수 표시 (복습 3개+ 시) → 앱 열지 않아도 인지 |
| S3-C | ⚙️ Engineer | **스플래시 화면 복습 알림** — 앱 실행 시 "오늘 복습할 단어 5개가 있어요!" 인터스티셜 |
| S3-D | 🎮 PM | **복습 우선 모드** — 홈 화면에 "지금 복습하기" / "새 게임 시작"을 같은 레벨로 배치 |

**가장 유망한 솔루션: S3-A** (복습 큐 데이터 이미 있음, UI 추가만 필요)

---

### O4: "streak이 끊기면 다시 시작하기 싫어진다"

**근본 원인:** streak 손실 = 실패 경험. 회복 메커니즘 없으면 재방문 포기 확률 급증.

| 솔루션 | 관점 | 설명 |
|--------|------|------|
| S4-A | 🎮 PM | **streak 회복 유예 (1일)** — streak 끊긴 다음 날 하루 내 플레이하면 streak 복구 ("오늘 복구 가능!") |
| S4-B | 🎨 Designer | **streak 끊김 위로 화면** — "괜찮아, 오늘부터 다시 시작하면 돼! 🔥" + 즉시 게임 시작 버튼 |
| S4-C | ⚙️ Engineer | **"주간 streak" 병행 추적** — 일간 streak 외 "이번 주 5/7일" 추적 → 실패해도 다른 성취 유지 |
| S4-D | 🎮 PM | **streak freeze (아이템)** — 배지로 "하루 유예권" 획득 가능 → streak 보험 |

**가장 유망한 솔루션: S4-A + S4-B 조합** (간단, 강력, 재방문 동기 직접 영향)

## Level 4: Experiments (검증 실험)

### Experiment 1: S1-B (배지 진행률 표시)

| 항목 | 내용 |
|------|------|
| **가설** | 결과 화면에 "다음 배지까지 N게임 남았어요!"를 표시하면, 같은 세션에서 두 번째 게임 시작률이 30% → 50%로 증가할 것이다 |
| **방법** | TestFlight A/B: 기존 결과 화면(A) vs. 배지 진행률 추가(B), 각 10명 |
| **측정 지표** | 결과 화면에서 "다시 하기" 버튼 탭률 |
| **성공 기준** | 탭률 A그룹 대비 B그룹 +20% 이상 |
| **소요 시간** | 1주 (구현 2시간, 측정 5일) |

### Experiment 2: S3-A (홈 화면 복습 배지)

| 항목 | 내용 |
|------|------|
| **가설** | 홈 화면에 "복습 N개" 카드를 표시하면, 복습 화면 진입 비율이 현재 추정 10% → 30%로 증가할 것이다 |
| **방법** | TestFlight 20명 관찰: 2주 전/후 복습 화면 진입 빈도 비교 |
| **측정 지표** | 세션당 복습 화면 진입 비율 |
| **성공 기준** | 복습 진입률 2배 이상 증가 |
| **소요 시간** | 2주 (구현 3시간, 측정 2주) |

### Experiment 3: S4-A (streak 회복 유예)

| 항목 | 내용 |
|------|------|
| **가설** | streak 끊김 다음 날 "오늘 복구 가능!" 메시지를 표시하면, streak 끊긴 후 D1 재방문율이 10% → 35%로 증가할 것이다 |
| **방법** | TestFlight에서 streak 끊긴 사용자 5~10명에게 유예 메시지 노출 → 다음 날 재방문 확인 |
| **측정 지표** | streak 끊긴 후 D1 재방문율 |
| **성공 기준** | D1 재방문율 3배 이상 증가 |
| **소요 시간** | 2주 (구현 4시간, 측정 2주) |
