# Ideal Customer Profile (ICP) — WordPang

**Date:** 2026-03-22
**Framework:** JTBD + 행동 프로파일링 (Clayton Christensen)
**데이터 기반:** 사용자 페르소나, 고객 여정 맵, OST 인터뷰 인사이트

## ICP 요약

> **WordPang의 ICP는 단일 개인이 아닌 "구매 결정권자(학부모) + 실제 사용자(아이)" 듀얼 프로파일이다.**

---

## 프로파일 A: 구매 결정권자 (학부모)

### 기본 인구통계

| 항목 | 이상적 ICP | 수용 가능 범위 |
|------|-----------|-------------|
| 연령 | 32~42세 | 28~48세 |
| 자녀 연령 | 8~10세 (초등 2~4학년) | 7~12세 (초등 전체) |
| 거주 지역 | 서울 / 수도권 | 전국 |
| 직업 | 맞벌이 또는 전업 (교육 관심도 높음) | 모두 |
| 교육 관심도 | 높음 (사교육 월 ₩30만+ 지출) | 중간 이상 |
| 스마트폰 활용 | 앱 설치·탐색에 익숙 | iOS 사용자 |

### 핵심 행동 패턴

**앱 발견 채널 (우선순위):**
1. 네이버 맘카페 "초등영어앱 추천" 검색 (가장 신뢰하는 채널)
2. App Store "어린이 영어" 카테고리 탐색
3. 학교 학부모 카톡방 추천
4. 유튜브 "엄마표 영어" 채널

**구매 결정 방식:**
- 무료 앱 → 즉시 설치, 아이에게 테스트 후 3~7일 내 지속 여부 결정
- 유료 앱 → 평점 4.5+, 리뷰 50개+, 스크린샷 훑어본 후 결정
- 개인정보 처리방침 탭 필수 확인 (데이터 수집 여부)

**현재 앱 경험:**
- 링고키즈: "3주 만에 질려서" → 구독 해지
- 리틀팍스: "영상 보는 것뿐이라 능동적 학습 아님"
- 유튜브 영어: "수동적이라 실제 학습인지 모르겠음"

| **감정적 직업** | "아이에게 좋은 교육을 제공하는 부모"라는 안도감을 느끼고 싶다 |

### 주요 Pain Points — 학부모

1. **반복 구매 피로**: "링고키즈 → 말해보카 → 리틀팍스... 계속 바꾸는 게 지친다"
2. **개인정보 우려**: "아이 이름과 학습 데이터를 앱에 넘기는 게 불안하다"
3. **학습 확인 어려움**: "앱 안에서 뭘 배웠는지 직접 확인해야 아는데 귀찮다"
4. **자발성 없음**: "시키면 하는데, 스스로 켜는 앱이 없다"
5. **광고 환경 우려**: "게임 중간에 광고 나오는 게 아이한테 노출시키기 싫다"

### 이상적인 ICP-of-the-ICP (최우선 타겟)

> **"초등 3학년 자녀를 둔 서울 거주 맞벌이 학부모, 링고키즈 이탈 후 대안 탐색 중, 개인정보 민감, 무료 앱 선호"**

- 이 프로파일이 App Store 검색 "초등영어 게임"에서 WordPang을 발견할 확률 최고
- 아이가 3일 내 자발적으로 앱을 열면 → 유지 + 맘카페 후기 + 카톡방 공유 트리거

## 프로파일 B: 실제 사용자 (아이)

### 기본 특성

| 항목 | 이상적 ICP | 수용 가능 범위 |
|------|-----------|-------------|
| 연령 | 8~10세 (초등 2~4학년) | 7~12세 |
| 영어 수준 | 초급-중급 (기초 단어 50~200개 아는 수준) | 입문~중급 |
| 게임 경험 | 모바일 게임 경험 있음 (규칙 학습 빠름) | 게임 조금이라도 경험 있음 |
| 집중 시간 | 7~12분 유지 가능 | 5분 이상 |

### 핵심 행동 패턴

- **첫 시도**: 혼자서 색깔 피드백(🟩🟧⬜) 의미 파악하려 시도
- **성공 경험 후**: 즉시 "다시 하기" 클릭 → 연속 플레이 경향
- **힌트 사용**: 전략적으로 아껴두거나, 점수 신경 안 쓰고 적극 사용 (양극화)
- **배지**: "이 배지 어떻게 받아요?" 질문 → 배지 리스트 자발적 탐색

### JTBD — 아이

| 직업 유형 | 구체적 내용 |
|---------|-----------|
| **기능적 직업** | "모르는 단어를 게임으로 알아내는 스릴을 즐기고 싶다" |
| **감정적 직업** | "어려운 단어를 맞혔을 때 '내가 해냈다!'는 성취감" |
| **사회적 직업** | "친구한테 '나 이 게임에서 2번 만에 맞혔어'라고 자랑하고 싶다" |
| **회피하는 직업** | 영어 단어 카드 암기 반복 → 지루함과 실패 감정 |

### 주요 Pain Points — 아이

1. **첫 번째 🟩 타일 이전**: "이 색깔이 뭔 뜻이야?" 혼란 → 빠른 온보딩 필요
2. **연속 실패**: 3~4번 틀리면 포기하고 싶어짐 → 힌트 유도 필요
3. **단조로움**: 같은 카테고리만 반복 → 카테고리 다양성 제공 필요
4. **Streak 끊김**: 하루 빠지면 "이미 끝났어" 포기 → 회복 메커니즘 필요

## ICP 활용 가이드

### 마케팅 메시지

**학부모 핵심 메시지:** "광고도, 구독도, 데이터 수집도 없어요. 아이가 스스로 켜는 영어 앱."

**아이 핵심 경험:** "색깔로 단어 맞히기 — 맞히면 짜릿해요!"

### 채널 우선순위

1. 네이버 맘카페 (학부모 가장 신뢰하는 정보 채널)
2. App Store ASO (구매 의사결정 순간)
3. 학교 학부모 카톡방 (최고 전환율 채널)
4. 유튜브 엄마표 영어 채널 (인지도 구축)

### 제품 의사결정에 ICP 적용

| 결정 상황 | ICP 기반 방향 |
|---------|------------|
| 광고 추가 여부 | ❌ 학부모 ICP가 가장 싫어하는 것 |
| 계정 시스템 추가 | ❌ 데이터 수집 우려 학부모 이탈 |
| 발음 오디오 추가 | ✅ 학부모 교육 가치 확인, 아이 학습 강화 |
| 무료 → 유료 전환 | ⚠️ MAU 2,000+ 리뷰 50개 후에만 — 그 전엔 설치 장벽 증가 |
