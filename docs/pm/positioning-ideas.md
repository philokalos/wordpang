# Positioning Ideas — WordPang

**Date:** 2026-03-22
**Framework:** 포지셔닝 갭 분석 + 차별화 포지셔닝 문장
**목표:** 한국 초등 영어 앱 시장에서 방어 가능한 포지션 확보

---

## Step 1: 경쟁사 포지셔닝 분석

| 경쟁사 | 핵심 포지셔닝 | 타겟 | 강조 차별점 | 열려있는 갭 |
|--------|------------|------|-----------|-----------|
| **링고키즈** | "전문 교육 전문가가 설계한 체계적 영어 학습" | 4~10세, 학부모 주도 | 콘텐츠 품질, 캐릭터 | "아이가 스스로 하는" 내재적 동기 없음 |
| **리틀팍스** | "영어 동화로 자연스럽게 영어 노출" | 3~13세, 스토리 중심 | 방대한 스토리 콘텐츠 | 능동적 학습 없음 (영상 소비) |
| **듀오링고** | "세계 1등 언어 학습 앱, 게임처럼 재미있게" | 성인~전연령 | 게임화, 글로벌 브랜드 | 초등 특화 없음, 광고 있음 |
| **말해보카** | "AI 기반 스마트 단어 학습" | 중학생+ | AI 기술, 효율적 암기 | 초등 저학년 UX 부적합 |
| **무료 유사앱** | "무료로 영어 공부" | 가격 민감 학부모 | 무료 | 품질·교육 가치 낮음, 광고 투성이 |

**포지셔닝 갭 발견:**
- **"자발적 학습" 포지션 미점유** — 경쟁사 모두 학부모가 시키는 학습 전제
- **"진짜 무료 + 진짜 광고 없음" 포지션 미점유** — 무료 앱은 광고, 유료 앱은 구독료
- **"초등 전용 추론 게임" 포지션 미점유** — Wordle 방식의 한국어 초등 특화 앱 없음
- **"데이터 수집 없는 안심" 포지션 미점유** — 모든 경쟁사가 데이터 수집

## Step 2 & 3: 5가지 포지셔닝 아이디어

### Positioning 1: "아이가 먼저 켜는 유일한 영어 앱"

**포지셔닝 문장:**
> WordPang은 영어 공부를 강요하지 않아도 되는, 아이가 스스로 열고 싶어하는 유일한 초등 영어 게임입니다.

**전략적 근거:**
- 학부모의 가장 깊은 욕구 = "시키지 않아도 스스로 하는 것"
- 경쟁사 모두 "좋은 교육"을 강조하지만 "자발성"을 포지셔닝 언어로 사용하지 않음
- Product Vision("아이가 먼저 하고 싶다고 말하는 날")을 마케팅 언어로 완전 연결

**서포팅 메시지:**
- "링고키즈는 3주, WordPang은 3개월 — 아이가 질리지 않아요"
- "게임이니까 공부처럼 느껴지지 않아요"
- "스스로 배지를 모으러 앱을 켜요"

**경쟁 우위 근거:**
- 배지 21개 + Streak + 간격 반복 = 내재적 동기 설계의 완결
- Wordle 방식 추론 게임 = 성취감 기반 학습 (강요 없는 능동성)
- 경쟁사 대비 검증 가능: "내 아이가 스스로 켰는가?" 는 학부모가 직접 경험

---

### Positioning 2: "영어 단어 암기의 반대말"

**포지셔닝 문장:**
> WordPang은 '외우는' 영어가 아닌 '발견하는' 영어를 위한 초등 단어 게임입니다.

**전략적 근거:**
- "암기 = 고통"이라는 학부모·아이 공통 페인 포인트 직격
- 링고키즈, 말해보카 모두 체계적 암기 커리큘럼 강조 → 반대 포지션 점유
- "발견"이라는 감정 언어가 WordPang의 추론 방식과 완벽 일치

**서포팅 메시지:**
- "'BRAVE' 맞혔어요! — 이 단어를 외운 게 아니라 발견했어요"
- "플래시카드 대신 추리 게임으로"
- "색깔 힌트로 스스로 알아내는 재미"

**경쟁 우위 근거:**
- Wordle 방식 = 정의상 "추론"이지 "암기"가 아님
- 학습 카드는 게임 후 자연스럽게 제공 (학습이 보상처럼 느껴짐)
- 간격 반복은 암기 부담 없이 자동 복습

---

### Positioning 3: "한국 초등 엄마들의 선택 — 광고 없는 영어 앱"

**포지셔닝 문장:**
> WordPang은 광고, 구독료, 데이터 수집 없이 초등학생이 진짜 즐기는 영어 게임입니다.

**전략적 근거:**
- 한국 맘카페 커뮤니티에서 "광고 없는 앱" = 신뢰의 최고 표시
- 무료 경쟁 앱들은 광고투성이, 유료 앱들은 구독료 — 이 조합을 부수는 포지션
- "엄마들이 선택한"은 사회적 증거(Social Proof) 강화 → 바이럴에 최적

**서포팅 메시지:**
- "링고키즈 ₩9,900/월 vs. WordPang 영구 무료"
- "아이 학습 데이터가 서버로 전송되지 않아요"
- "광고를 이기려면 돈을 쓰거나 나쁜 앱을 만들어야 한다 — WordPang은 둘 다 안 해요"

**경쟁 우위 근거:**
- 오프라인 아키텍처 = 서버 비용 없음 → 무료 지속 가능
- COPPA/PIPA 회피 = 법적으로 데이터 수집 없음 증명 가능
- 1인 개발자 진솔한 스토리 = 기업 마케팅 대비 신뢰도 높음

---

### Positioning 4: "초등 영어의 Wordle — 하루 한 게임"

**포지셔닝 문장:**
> WordPang은 Wordle의 추론 재미를 초등학생 영어로 가져온, 하루 5분으로 충분한 단어 게임입니다.

**전략적 근거:**
- Wordle 문화(2022~)는 전 세계적으로 알려진 브랜드 패턴 → 즉각 이해
- "하루 한 게임"은 부담 없는 습관 형성 포지셔닝 → 장기 리텐션 약속
- 경쟁사 모두 긴 학습 시간(30분+)을 암묵적 전제 → 시간 부담 없음 차별화

**서포팅 메시지:**
- "전 세계가 즐긴 Wordle 방식으로 영어 단어를"
- "하루 5분, 한 단어 발견"
- "짧지만 강력한 — 매일 하는 힘"

**경쟁 우위 근거:**
- 게임 구조가 자연스럽게 하루 1~3게임으로 설계됨
- Streak 메커니즘이 "매일 조금씩" 습관을 강화
- 시간 부담 없음 = 학부모의 "스마트폰 과의존 우려" 완화

---

### Positioning 5: "초등 영어 내 이름을 자랑할 수 있는 게임"

**포지셔닝 문장:**
> WordPang은 배지를 모으고, 결과를 자랑하고, 친구에게 도전할 수 있는 초등 영어 성취 게임입니다.

**전략적 근거:**
- 초등학생의 핵심 동기 = 또래와의 비교, 자랑
- "공부 결과를 공유"가 아닌 "게임 기록을 자랑" → 학습이 문화가 됨
- 카카오톡 이모지 공유 🟩🟧⬜는 Wordle 공유 문화와 완벽 연결 → 바이럴 루프

**서포팅 메시지:**
- "2번 만에 맞혔어! 이모지 공유하기 🟩🟩🟩"
- "배지 21개 중 내 친구는 몇 개 모았을까?"
- "같은 단어, 같은 도전"

**경쟁 우위 근거:**
- 이모지 공유 기능 이미 구현됨 (강화 필요)
- 21개 배지 = 자랑하고 싶은 수집 아이템
- 동일 단어 일일 챌린지 → 친구 비교 자연 발생

## 최종 포지셔닝 권고

### 현재 단계 (출시~MAU 2,000): **Positioning 3 주력**
> "광고 없는 진짜 무료 초등 영어 게임"

- 이유: App Store 검색 + 맘카페 진입 시 즉각적 신뢰 형성
- 가장 검증 가능하고 경쟁사가 가장 복사하기 어려운 포지션

### Phase 2 (MAU 2,000+): **Positioning 1 강화**
> "아이가 먼저 켜는 영어 앱"

- 리뷰 50개+ 이후 실제 학부모 경험이 쌓이면 "아이가 혼자 켰어요" 후기로 증거 강화
- App Store 설명, 맘카페 캠페인에서 Positioning 1으로 전환

### 복합 포지셔닝 문장 (두 가지 통합):
> **"광고도 구독도 없는 완전 무료 — 아이가 스스로 열고 싶은 초등 영어 단어 게임"**
