## 4기준 평가 매트릭스

**평가 기준 (각 1~5점)**
1. **Burning Pain**: 문제가 얼마나 절박한가?
2. **Willingness to Pay**: 비용을 지불할 의향/능력이 있는가? (현재는 무료이므로 앱 설치 의향)
3. **Winnable**: 3~18개월 내 60~70% 점유 가능한가?
4. **Referral Potential**: 추천 경로와 확장성이 있는가?

## 세그먼트별 상세 분석

### S1: 초등 3~4학년 학부모 ★ 추천 비치헤드

**Burning Pain (4/5)**
- 초등 3학년부터 영어 수업 시작 → 학부모의 첫 영어 앱 탐색 시점
- "링고키즈 3주 만에 질려서 그만뒀는데" — 학습 지속성 문제 절박
- 단어 암기보다 "영어를 좋아하게 만들고 싶다" 욕구 강함

**WTP/설치 의향 (4/5)**
- 무료 앱은 즉시 설치 → 전환 장벽 없음
- 이 연령 학부모는 교육 앱 탐색에 가장 적극적 (영어 시작 단계)
- 유료 전환 시 "$1.99 = 링고키즈 3일치" 논리 수용 가능

**Winnable (4/5)**
- 초등 3~4학년 영어 앱 = 링고키즈, 리틀팍스, 무료 유사 앱 경쟁
- WordPang의 "무료 + 게임형 + Wordle 방식" = 이 세그먼트에서 미개척 포지션
- 맘카페 + App Store ASO로 3개월 내 200~500 다운로드 달성 가능

**Referral Potential (5/5)**
- 학부모 카톡방 ("우리 반 학부모 단톡") → 앱 추천 폭발적 확산 가능
- "아이가 혼자서 앱 켰어요!" → 맘카페 자발적 후기 → 바이럴
- S2 (5~6학년), S3 (교사) 확장 경로 명확

---

### S2: 초등 5~6학년 학부모

**Burning Pain (5/5)** — 가장 높음
- 단어 암기 숙제 + 영어 시험 부담 → 절박함 최고
- "단어 외우라고 하면 애가 짜증내요" — 정확히 WordPang이 해결하는 문제

**하지만 Winnable이 낮은 이유 (3/5)**
- 이 연령은 "재미" 외 "성적 향상"도 증명 필요 → 학원 앱과도 경쟁
- WordPang이 단기 시험 대비 단어 커리큘럼을 아직 보유하지 않음 (514단어가 교과서 연계 필요)

**S1 달성 후 자연스러운 확장 세그먼트 (Phase 2)**

---

### S3: 초등 영어 교사

**Referral Potential이 매우 높은 이유 (5/5)**
- 교사 1명 = 학생 20~30명 + 학부모 20~30명에게 추천 가능
- 인디스쿨 교사 커뮤니티 → 1게시물이 수천 명 교사에게 도달

**하지만 WTP가 낮은 이유 (3/5)**
- 교사 개인 자비 지출은 어려움
- 학교 예산 집행은 행정 절차 복잡 → 1인 개발자에게 B2B 영업 부담 과다
- S1 비치헤드 성공 후 Phase 3로 공략

## 🎯 최종 비치헤드 세그먼트

### **초등 3~4학년 아이를 둔 학부모 (30~40대, 서울·수도권 거주)**

**왜 이 세그먼트가 최우선인가:**

1. **영어 학습 시작점** = 첫 번째 앱 선택 시점 → 습관 형성의 골든 윈도우
2. **대안에 지친 상태** = 링고키즈 3주 후 이탈 경험 → WordPang 차별점에 가장 민감
3. **학부모 카톡방 네트워크** = 입소문 확산 밀도 최고 (동질 집단)
4. **무료 설치 장벽 없음** = 탐색적 설치 후 아이 반응으로 결정
5. **S2·S3 확장 경로 직접 연결** = 비치헤드 성공 후 확장이 자연스러움

## 비치헤드 공략 90일 계획

### Day 1~30: 첫 신뢰 구축
- App Store 최적화 — 키워드: "초등영어 게임", "영어단어 어린이", "무료 영어 앱"
- 스크린샷 순서: 1. 게임 장면 (아이 흥미) → 2. 통계 화면 (학부모 신뢰) → 3. 무광고 강조
- TestFlight 베타로 맘카페 선발대 10명 모집

### Day 31~60: 구전 활성화
- 맘카페 개발자 진솔한 소개글 1편 (광고 아닌 "왜 만들었나" 스토리)
- 배지 달성 시 "공유하기" CTA 구현 → 카카오톡 자연스러운 전파
- App Store 리뷰 20개 달성 목표

### Day 61~90: 비치헤드 공략 완성
- 리뷰 50개 + DAU 200+ 달성 평가
- S2 (5~6학년) 확장 신호 탐지 — App Store 리뷰에서 나이 언급 패턴 분석
- S3 (교사) 채널 1차 접근 (인디스쿨 탐색)
