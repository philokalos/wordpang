# Lean Canvas — WordPang

**Date:** 2026-03-22
**Stage:** Pre-launch (App Store 제출 준비 중)

## Section 1: Product Definition

### 1. Problem

**Top 3 고객 문제:**

| # | 문제 | 현재 불만족스러운 해결책 |
|---|------|----------------------|
| P1 | 단어 암기 앱은 3일 만에 지루해서 삭제한다 | 단어장 앱, 교과서 부록 — 반복만 있고 피드백 없음 |
| P2 | 교육 효과가 있는 앱은 학부모 구독료 부담이 크거나 광고로 가득 차 있다 | 링고키즈(₩9,900/월), 듀오링고(광고 또는 연 ₩99,000) |
| P3 | 배운 단어를 금방 잊어버린다 (에빙하우스 망각 곡선) | 앱 대부분이 배운 단어를 체계적으로 재학습시키지 않음 |

---

### 2. Solution

| # | 솔루션 | 해결하는 문제 |
|---|--------|------------|
| S1 | 색깔 힌트 기반 단어 추리 게임 (퍼즐 메커니즘) | P1 — 맞히는 재미로 내재적 동기 유발 |
| S2 | 무료·무광고·무데이터 수집 구조 | P2 — 학부모 신뢰 획득, 진입 장벽 제거 |
| S3 | 자동 간격 반복 복습 (30min × 2ⁿ 알고리즘) | P3 — 에빙하우스 망각 곡선 대응, 장기 기억 전환 |

---

### 3. Unique Value Proposition (UVP)

> **"외우지 말고 맞혀보세요 — 초등 영어 단어를 발견하는 게임"**

- **차별화 (vs. 암기 앱)**: 수동적 암기 → 능동적 추리로 전환 → 정보처리 깊이 증가
- **차별화 (vs. 유료 앱)**: 완전 무료 + 무광고 → 학부모 채택 장벽 없음
- **차별화 (vs. 듀오링고)**: 영어 전반이 아닌 어휘 특화 + 교정 피드백 게임

---

### 4. Unfair Advantage

| 방어 요소 | 설명 |
|----------|------|
| **무데이터 정책** | 서버 없는 구조는 단순한 기능이 아닌 아키텍처적 약속 — 유료화/데이터 수집으로 쉽게 전환 불가 |
| **교육 연구 기반 설계** | 4개 검증된 이론 통합 적용 — 단순 클론으로 효과 재현 불가 |
| **514개 단어 큐레이션 자산** | 뜻·발음·예문 수작업 검수 — 복제 시 동일 노력 필요 |
| **App Store 리뷰 자산** | 출시 후 축적되는 리뷰·평점 — 신규 진입자 불리 |

## Section 2: Market & Traction

### 5. Customer Segments

**주 세그먼트: 단어 학습이 지루한 초등학생 (8~12세)**

- 스마트폰 보유, 게임 친숙, 집중력 5~15분
- Early Adopter: 영어 흥미 있는 초등 3~4학년, 학원 다니는 아이

**보조 세그먼트: 교육 앱 선택권 가진 학부모**

- 구독료 부담, 광고·개인정보 거부감
- Early Adopter: 교육 블로그 팔로우하는 30~40대 학부모

**시장 규모 추산 (한국 기준):**

| 단계 | 규모 | 근거 |
|------|------|------|
| TAM | ~500만 명 | 초등학생 전체 (2025년 기준 약 267만 → 스마트폰 보유 포함 500만) |
| SAM | ~150만 명 | 영어 학습 앱 사용 경험 있는 초등학생 |
| SOM | ~5만 명 | App Store 유기적 검색으로 도달 가능한 1년 내 목표 |

### 7. Revenue Streams

**현재 모델: 완전 무료 (수익 없음)**

| 항목 | 현재 | 미래 옵션 |
|------|------|---------|
| 앱 가격 | $0 | $0.99~$2.99 유료 전환 가능 |
| 인앱 결제 | 없음 | 추가 단어 팩 ($0.99) |
| 구독 | 없음 | 교사용 클래스 관리 ($4.99/월) |
| B2B | 없음 | 학교/학원 라이선스 |

**현재 전략적 선택:** 무료로 MAU 최대화 → App Store 리뷰·평점 축적 → 이후 가격 책정 옵션 검토

**LTV 추산 (무료 기준):** 직접 수익 없음. 간접 가치 = 브랜드 신뢰 + 후속 제품 기반
