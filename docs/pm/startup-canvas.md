## Part 1: Product Strategy

### 1. Vision

> **"아이가 '영어 단어 게임 하고 싶어요'라고 먼저 말하는 날을 만든다"**

English: *"Make every child say 'I want to play' — not 'I have to study.'"*

- **Inspire**: 외재적 압박(점수, 숙제) → 내재적 동기(발견의 기쁨)로 패러다임 전환
- **Achieve**: 게임 기반 학습 + 간격 반복으로 장기 기억 전환 — 이미 구현 중
- **Values**: 무료·무광고·무데이터 수집 (아이의 놀이를 상업적으로 착취하지 않는다)

---

### 2. Market Segments

**시장 = 사람들의 문제로 정의됨 (인구통계가 아님)**

| 세그먼트 | JTBD (Job-to-be-Done) | Desired Outcome | Constraints |
|---------|----------------------|-----------------|-------------|
| **초등학생 (7~12세)** | "영어 단어를 암기 고통 없이 재미있게 배우고 싶다" | 수업·시험에서 단어가 떠오른다 | 집중 시간 짧음, 어려우면 즉시 이탈 |
| **학부모 (30~45세)** | "아이가 자발적으로 열어서 영어를 자연스럽게 익히는 앱을 찾고 싶다" | 아이의 학습 진도 확인 + 안심 | 개인정보 수집·광고에 민감, 월정액 부담 |

**첫 번째 세그먼트**: 학부모 (구매 결정권자) + 초등학생 (실사용자)
**이유**: 학부모의 App Store 검색 → 설치 → 아이에게 건네주는 듀얼 여정이 핵심 채널

**TAM / SAM / SOM (한국)**
| 수준 | 규모 | 기준 |
|------|------|------|
| TAM | 500만 명 | 한국 초등학생 전체 |
| SAM | 150만 명 | 스마트폰 보유 + 영어 학습 앱 사용 의향 |
| SOM | 5만 명 | 출시 후 18개월 내 달성 목표 DAU |

### 3. Relative Costs

**포지션: Premium Value at Zero Cost**

- 가격 = $0 (무료) → 진입 장벽 최소화, 시장 점유 우선
- 품질 = 스케치 디자인 시스템, 간격 반복, 21개 배지, 514개 단어 — 유료 앱 수준
- 경쟁사(링고키즈 ₩9,900/월, 리틀팍스 ₩6,900/월) 대비 **압도적 가성비 차별화**
- 수익화는 MAU 2,000+ 이후 S1(일회성 유료 $1.99) 전환 검토

### 4. Value Proposition

#### 초등학생용
| 항목 | 내용 |
|------|------|
| **What Before** | 단어 카드 암기 반복 → 지루함, 실패 → 영어 싫증 |
| **How** | Wordle 방식 추론 게임 + 색깔 피드백 + 힌트 전략 + 배지 보상 |
| **What After** | "내가 맞혔다!" 성취감 → 내재적 동기 → 자발적 재방문 |
| **Alternatives** | 플래시카드(지루), 링고키즈(비싸), 유튜브 영상(수동적) |

#### 학부모용
| 항목 | 내용 |
|------|------|
| **What Before** | 비싸거나 광고 많은 앱, 데이터 수집 우려, 아이가 며칠 후 그만 |
| **How** | 완전 무료·무광고·무데이터 + 통계 화면으로 학습 진도 확인 |
| **What After** | 아이가 혼자서 앱 켜는 장면 + "이번 달 47단어 학습 완료" 확인 |
| **Alternatives** | 링고키즈(₩9,900/월), 학원(비싸고 의존적), 방치 |

### 5. Trade-offs (우리가 하지 않는 것)

| 우리가 하지 않는 것 | 이유 |
|-------------------|------|
| 광고 삽입 | 브랜드 자산(무광고) 훼손이 수익보다 비용이 크다 |
| 서버/계정 시스템 | 개인정보 리스크 + COPPA/PIPA 부담 + 오프라인 차별점 약화 |
| Android 동시 출시 | 1인 개발자 — iOS 한 플랫폼 완성도 집중 |
| 경쟁사 기능 따라하기 | 구독 모델/광고 없는 포지션이 핵심 차별점 |
| 멀티플레이어 | 경쟁이 내재적 동기 훼손 + COPPA 복잡도 |
| 5세 이하 타겟 | UX 요구사항 상이 → 집중도 저하 |

### 6. Key Metrics

**North Star Metric**: Words Mastered per MAU
- Mastered = 간격 반복 3회 완료 (30분 → 1시간 → 2시간 주기)
- 왜: 장기 기억 전환 = 내재적 동기의 결과 = 비전 달성의 직접 증거

**OMTM Q2 2026**: D3 Retention > 30%
- 현재 기준값: 미측정 (출시 전)
- 목표: 25% → 30% (경쟁 교육 앱 평균 D3 ~20%)
- 왜 D3: 재방문 없이는 복습 루프도, mastered 단어도, NSM도 없다

**Supporting Metrics**:
- D1 Retention (첫날 재방문율)
- Session Length (7~10분 목표)
- Hint Usage Rate (힌트 사용률 → 인게이지먼트 프록시)
- Badge Completion Rate (배지 달성율 → 몰입도)
- Review Completion Rate (복습 완료율 → 간격 반복 효과)

### 7. Growth

**모델: Product-Led Growth (PLG)**

1단계 — App Store 진입 (현재)
- ASO: "초등 영어 게임" "어린이 영어 앱" "영어 단어 게임" 최적화
- 키워드: Wordle + 초등 + 영어 단어 + 무료

2단계 — 바이럴 루프 (출시 후)
- 결과 이모지 공유 🟩🟧⬜ → 카카오톡 공유 → 친구 다운로드
- 학부모 커뮤니티 (맘카페) 입소문 → 비용 없는 오가닉 채널

3단계 — 선생님 채널 (MAU 2,000+)
- 인디스쿨 교사 커뮤니티 → B2B2C 채널 개발

**채널 우선순위**:
| 채널 | 비용 | 기대 효과 | 우선순위 |
|------|------|---------|---------|
| App Store ASO | $0 | 검색 발견성 | 최고 |
| 맘카페 (네이버/다음) | $0 | 신뢰 기반 구전 | 높음 |
| 이모지 공유 | $0 | 바이럴 루프 | 중간 |
| 교사 커뮤니티 (인디스쿨) | 시간 비용 | 클래스 단위 확산 | 나중 |

### 9. Can't/Won't (전략 방어성)

**경쟁사가 이 전략을 복사할 수 없는 이유:**

| 테스트 | 이유 |
|--------|------|
| **Can't** | 듀오링고 등 대형사는 광고/구독 수익 모델에 묶여 있어 완전 무료 포지션을 취할 수 없다 (수익 모델 충돌) |
| **Won't** | 링고키즈/리틀팍스는 구독 MAU 기반이므로 오프라인·무데이터 방향 전환 시 기존 과금 구조 붕괴 |
| **Won't** | 대형사는 초등 특화 Sketchy 디자인 시스템과 한국어 문화 맥락 적용을 우선하지 않음 |

**전략적 일관성 체크**:
무료 → 오가닉 사용자 → 단어 Mastered 증가 → 학부모 신뢰 → 맘카페 입소문 → 더 많은 오가닉 사용자. 광고/구독을 추가하면 이 루프 전체가 깨진다.

### 11. Revenue Streams

**현재: $0 (완전 무료)**

**로드맵:**

| 단계 | 조건 | 전략 | 예상 수익 |
|------|------|------|---------|
| Phase 1 (지금~MAU 2,000) | 출시 후 90일 | 완전 무료 유지 | $0 |
| Phase 2 (MAU 2,000+) | 리뷰 50개+ | S1: 일회성 $1.99 전환 테스트 | 월 500 DL × $1.99 × 0.7 = ~$700/월 |
| Phase 3 (MAU 5,000+) | 검증 결과 기반 | S2: 콘텐츠 팩 또는 S4: 앱 생태계 | $500~$1,000/월 |

**수익 모델 선택 기준**:
- 광고 수익: ❌ 절대 금지 (브랜드 자산 훼손)
- 구독: ⚠️ 신중 (무료 차별점 약화)
- 일회성 유료: ✅ 브랜드 일관성 유지하며 수익화 가능
- B2B 라이선스: ✅ MAU 확보 후 고LTV 채널
