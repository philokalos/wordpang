# User Personas — WordPang

**데이터 기반:** PRD 사용자 세그먼트 분석, 교육 연구 문헌, 경쟁 앱 리뷰 패턴 분석

## Persona 1: "게임 좋아하는 초등학생" — 지호 (9세, 초등 3학년)

### 프로필
| 항목 | 내용 |
|------|------|
| 나이 | 9세 |
| 학년 | 초등 3학년 |
| 기기 | 부모 구 스마트폰 물려받음 (iPhone SE) |
| 영어 수준 | 알파벳은 완벽, 기초 단어 50~100개 아는 수준 |
| 특성 | 마인크래프트, 쿠키런 즐김; 집중력 10~15분; 게임에서 이기고 싶은 승부욕 강함 |

### Primary JTBD
> "게임을 하는 것 같은데 학원 선생님이 깜짝 놀라도록 단어를 많이 알고 싶다."

- **컨텍스트:** 학원 단어 시험이 매주 있음. 시험 전날 외우면 당일만 기억. 게임을 하면서 자연스럽게 익힌 단어는 오래 기억됨.
- **빈도:** 매일 10~15분, 주로 저녁 식사 후 또는 잠자리 들기 전

### Top 3 Pain Points
1. **단어 암기가 너무 지루하다** — 단어장을 5분 보면 잠이 온다. 억지로 외워도 시험이 끝나면 다 잊어버린다.
2. **어려운 단어는 포기하게 된다** — 5글자 이상 단어는 처음부터 외울 엄두가 안 난다. 힌트가 없으면 막막하다.
3. **잘하고 있는지 모른다** — 얼마나 배웠는지, 어디까지 왔는지 알 수 없다. 성취감이 없다.

### Top 3 Desired Gains
1. **"내가 맞혔다!"는 성취감** — 정답을 맞히는 순간의 짜릿함. 🟩이 다 채워질 때의 만족감.
2. **배지와 연속 기록** — 🔥7일 연속!, 🏆노힌트 챔피언 같은 배지를 모으는 재미.
3. **학원 시험에서 아는 단어가 늘어나는 경험** — "어? 이 단어 WordPang에서 봤어!" 하는 순간.

### One Unexpected Insight
> **힌트를 쓰는 것 자체가 학습이다** — 아이들은 힌트를 "실패의 표시"로 보지 않는다. 오히려 "전략"으로 사용한다. "이 힌트 쓰면 포인트 깎이니까 먼저 추리해볼게"라는 사고 과정이 더 깊은 단어 처리를 유도한다.
>
> → **제품 시사점:** 힌트 패널을 "부끄러운 도움"이 아닌 "전략 도구"로 포지셔닝할 것. 힌트 사용 패턴 분석으로 어떤 힌트가 가장 학습 효과가 높은지 파악 가능.

### Product Fit Assessment
| 요소 | 적합도 | 메모 |
|------|--------|------|
| 퍼즐 메커니즘 | ●●●●● | 게임을 좋아하는 지호에게 최적 |
| 배지 시스템 | ●●●●● | 수집욕 강한 아이에게 강력한 동기 |
| 난이도 조절 | ●●●●○ | Easy부터 시작해서 점진적 도전 가능 |
| 발음 오디오 (미구현) | ●●●○○ | 영어 발음을 소리로 확인하고 싶어함 |
| 사회적 요소 없음 | ●●○○○ | 친구와 비교하거나 보여주고 싶은 욕구 미충족 |

## Persona 2: "성취지향 학생" — 서연 (11세, 초등 5학년)

### 프로필
| 항목 | 내용 |
|------|------|
| 나이 | 11세 |
| 학년 | 초등 5학년 |
| 기기 | 본인 스마트폰 (iPhone 15) |
| 영어 수준 | 중급 — 기초 문장 구성 가능, 어휘 300개+ |
| 특성 | 공부 잘하려는 동기 강함; 영어 학원 다님; 완벽주의 성향; 목표 설정·달성에서 만족 |

### Primary JTBD
> "Hard 모드를 완전히 정복하고, 영어 어휘를 체계적으로 늘리고 싶다."

- **컨텍스트:** 학원에서 배운 단어를 더 깊이 이해하고, 영어 시험에서 상위권에 있고 싶다.
- **빈도:** 매일 15~20분; 복습 알림이 오면 바로 확인하는 타입

### Top 3 Pain Points
1. **배운 단어를 체계적으로 복습할 방법이 없다** — 단어는 많이 아는데 어떤 걸 더 복습해야 하는지 모른다. 앱이 알아서 스케줄을 잡아줬으면 한다.
2. **카테고리별로 진도를 확인하고 싶다** — "동물 단어는 다 아는 것 같은데 감정 단어는 약한 것 같아" — 이런 분석을 해보고 싶다.
3. **Hard 모드가 너무 어렵거나 너무 쉽다** — 적응형 난이도가 내 현재 실력을 정확히 읽어주길 바란다.

### Top 3 Desired Gains
1. **체계적인 진도 관리** — "이번 달 몇 단어 마스터했다"는 숫자로 보이는 성장.
2. **약점 카테고리 파악** — "feeling 카테고리가 제일 약하다" → 집중 연습.
3. **Hard 모드 완전 정복 배지** — 어려운 도전을 완수했다는 증명.

### One Unexpected Insight
> **서연은 앱을 "자기주도 학습 도구"로 사용하고 싶다** — 부모나 선생님에게 "이것 써"라는 권유보다, 스스로 선택하고 통제하는 느낌을 원한다. 통계 화면을 자주 보고, 자기가 얼마나 잘하고 있는지 데이터로 확인하는 것을 즐긴다.
>
> → **제품 시사점:** 통계 화면의 자세함과 카테고리별 분석 깊이가 서연 같은 성취지향 사용자의 재방문을 유도하는 핵심 요소다. "약점 카테고리 집중 모드" 기능이 고가치 P2 아이템.

### Product Fit Assessment
| 요소 | 적합도 | 메모 |
|------|--------|------|
| 간격 반복 복습 | ●●●●● | 서연의 체계적 학습 니즈에 완벽 부합 |
| 카테고리별 진도 | ●●●●● | 통계 화면에서 이미 제공 |
| 적응형 난이도 | ●●●●○ | 있으나 자동 추천 수준 — 더 정밀한 제어 원함 |
| 복습 큐 우선순위 | ●●●○○ | 어떤 단어를 먼저 복습해야 하는지 설명 필요 |
| 약점 분석 | ●●●○○ | 카테고리별 정확도 분석은 미구현 |

## Persona 3: "교육 앱 선별자" — 지은 (38세, 초등 3학년·5학년 두 아이 엄마)

### 프로필
| 항목 | 내용 |
|------|------|
| 나이 | 38세 |
| 직업 | 마케터 (재택근무) |
| 자녀 | 9세 (3학년), 11세 (5학년) |
| 정보 채널 | 네이버 맘카페, 인스타그램 교육 계정, 유튜브 "엄마표 영어" |
| 특성 | 교육에 적극적 투자; 앱 구독 3개 이상 동시 사용 경험; 구독 피로도 높음 |

### Primary JTBD
> "두 아이에게 동시에 줄 수 있는 영어 단어 앱을 찾고 싶다 — 광고 없고, 개인정보 안 모으고, 실제로 효과 있는."

- **컨텍스트:** 링고키즈 2개 구독 중 (아이별) → 월 ₩19,800; 아이들이 3~4주 후 흥미 잃음.
- **빈도:** 앱 평가는 1~2주 사용 후; 좋은 앱은 맘카페에 공유하는 인플루언서 행동

### Top 3 Pain Points
1. **구독 앱은 아이들이 금방 흥미를 잃는다** — ₩9,900 × 2 = ₩19,800/월인데 아이들이 알아서 열지 않는다. 잔소리해서 쓰게 하는 건 의미 없다.
2. **광고가 있으면 바로 삭제한다** — 아이가 앱 쓰다가 광고 팝업이 뜨면 바로 지워버린다. 특히 게임 광고, 인앱 결제 유도가 제일 싫다.
3. **개인정보 수집 동의 팝업이 불안하다** — "아이 데이터가 어디로 가는지 모르겠어"라는 걱정. 특히 이름·나이·학년 입력 요구 시 즉시 거부.

### Top 3 Desired Gains
1. **아이가 스스로 앱을 켜는 경험** — "엄마, WordPang 하면 돼요?" — 이 한 마디가 최고의 성공.
2. **학습 효과의 가시성** — 통계 화면에서 "이번 달 단어 47개 학습"을 보고 공유하고 싶음.
3. **맘카페에 추천할 수 있는 앱** — "우리 애 영어 단어 앱인데 완전 무료에 광고도 없어요. 근데 진짜 효과 있더라고요." — 이 말을 할 수 있는 앱.

### One Unexpected Insight
> **지은은 "교육 효과"보다 "지속 사용 가능성"을 먼저 평가한다** — 아무리 좋은 교육 앱도 아이가 안 열면 소용없다는 것을 경험으로 알기 때문이다. 앱을 선택할 때 첫 번째 기준은 "우리 아이가 스스로 계속 쓸까?"이고, 교육 효과는 두 번째다.
>
> → **제품 시사점:** 학부모 마케팅 메시지는 "교육 효과 494%"보다 "아이가 혼자서 열어요"를 앞에 배치할 것. App Store 스크린샷 1번에 게임 장면, 마지막에 통계 화면이 올바른 순서.

### Product Fit Assessment
| 요소 | 적합도 | 메모 |
|------|--------|------|
| 무료·무광고 | ●●●●● | 지은의 최우선 기준 충족 |
| 무데이터 수집 | ●●●●● | COPPA·PIPA 완전 준수로 불안 해소 |
| 두 아이 동시 사용 | ●●●●○ | 기기별 독립 저장 → 계정 없이 각각 사용 가능 |
| 통계 화면 | ●●●●○ | 학습 가시성 있으나 부모 뷰 별도 없음 |
| 맘카페 공유 소재 | ●●●●● | 이모지 결과, 무료·무광고 메시지가 공유 소재 |
| 형제 프로필 분리 | ●●○○○ | 단일 기기에서 두 아이 프로필 분리 미지원 — 미래 기능 |
