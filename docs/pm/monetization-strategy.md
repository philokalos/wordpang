# Monetization Strategy — WordPang

**Date:** 2026-03-22
**현재 모델:** 완전 무료 (수익 없음)
**전제 조건:** 단기적으로 무료 포지션 유지가 핵심 차별점. 수익화는 MAU 5,000+ 이후 검토.

---

## 시장 배경

| 경쟁사 | 모델 | 가격 |
|--------|------|------|
| 듀오링고 | Freemium + 구독 | 무료 / $6.99/월 |
| 링고키즈 | 구독 | ₩9,900/월 |
| 말해보카 | Freemium + 구독 | 무료 / ₩4,900/월 |
| 리틀팍스 | 구독 | ₩6,900/월 |

**기회 공간:** 경쟁사 모두 구독 또는 광고 기반. "일회성 저가 유료" + "B2B2C 교사 채널"은 미개척.

## Strategy 1: 일회성 유료 전환 (One-Time Purchase)

### 설명
현재 완전 무료 → 앱 가격 $0.99~$2.99 유료 전환 (기존 기능 유지, 추가 콘텐츠 없음)

### How It Works
- App Store에서 $1.99 유료 앱으로 전환
- 출시 첫 달 런칭 프로모션 → 무료 또는 $0.99
- "한 번 구매 = 영구 사용, 업데이트 포함"
- 광고·인앱 결제 없음 유지

### Audience Fit
| 항목 | 평가 |
|------|------|
| 학부모 지불 의향 | 높음 — 교육 앱 $1.99는 커피 한 잔보다 저렴 |
| 초등학생 직접 구매 | 낮음 — 학부모 결정 |
| 경쟁 대비 | 링고키즈 월 ₩9,900 vs. 일회성 $1.99 = 압도적 가성비 |

**잠재 수익 (연간):**
- 월 500 다운로드 × $1.99 × 12개월 = $11,940/년
- Apple 30% 수수료 후 → ~$8,350/년

### Unit Economics
| 항목 | 값 |
|------|-----|
| CAC | $0 (유기적) |
| LTV | $1.39 (Apple 수수료 후) |
| 손익분기 | 즉시 (서버 없음) |
| 목표 마진 | ~70% |

### Risks
- 무료 → 유료 전환 시 다운로드 급감 (전형적으로 80~90% 하락)
- 현재의 핵심 차별점("완전 무료")이 약화
- 출시 초기에 가격 설정 시 초기 리뷰 확보 어려움

### Validation Experiment
- TestFlight 100명에게 "$1.99이면 구매하겠나?" 인앱 설문
- 성공 기준: 응답자의 40%+ "예"
- 소요 시간: 출시 후 30일

## Strategy 2: 콘텐츠 팩 인앱 결제 (In-App Purchase)

### 설명
앱은 무료 유지 (기본 120단어/난이도 = 360개), 추가 단어 팩을 $0.99~$1.99에 판매

### How It Works
- **무료 기본:** Easy/Normal/Hard 각 ~60단어 (총 180개)
- **프리미엄 팩 (각 $0.99):**
  - "학교 영어 팩" — 교과서 필수 단어 100개
  - "동물 특화 팩" — 동물 단어 심화 50개
  - "감정 표현 팩" — 감정·행동 단어 심화 50개
  - "전체 팩" — $2.99 번들 (현재 514개 모두)

### Audience Fit
| 항목 | 평가 |
|------|------|
| 학부모 지불 의향 | 중간 — 특정 필요(학원 시험 대비)에는 구매 의향 |
| 전환 가능 사용자 비율 | 낮음 — 무료로 충분하다고 느끼면 구매 안 함 |
| 자녀 교육 투자 심리 | 긍정적 — "학교 영어 시험 팩"에 특화 |

**잠재 수익:**
- MAU 5,000 × 전환율 3% × $1.99 = 월 $298
- 번들 구매 포함 월 $500~800 추산

### Unit Economics
| 항목 | 값 |
|------|-----|
| CAC | $0 |
| LTV | $2~5 (팩 1~3개 구매) |
| 전환율 목표 | 3~5% (교육 앱 평균 2~4%) |

### Risks
- 무료 단어로도 충분히 재미있으면 유료 전환 인센티브 없음
- "학원 교재 연계" 없이는 "학교 영어 팩" 가치 설명 어려움
- 콘텐츠 팩 관리 복잡도 증가

### Validation Experiment
- 홈 화면에 "프리미엄 단어 팩 미리보기" 버튼 추가
- 클릭률 측정 (전체 사용자 중 5%+ 클릭 시 수요 있음)
- 소요 시간: 출시 후 60일

## Strategy 3: 교사·학교 B2B 라이선스 (B2B2C)

### 설명
초등 영어 교사 대상 클래스 관리 기능 + 단체 라이선스 판매

### How It Works
- **교사 앱 (월 $4.99~$9.99):**
  - 학생 진도 모니터링 대시보드 (선생님용)
  - 클래스 단어 과제 지정 기능
  - 학생 단어 달성 리포트 (주간 자동 생성)
- **학교 라이선스 (연간 ₩50,000~₩200,000/반):**
  - 교사 1명 + 학생 30명 묶음
- **학생 앱은 계속 무료** (광고·구독 없음 유지)

### Audience Fit
| 항목 | 평가 |
|------|------|
| 교사 지불 의향 | 중간 — 학교 예산 사용 가능하면 구매 결정 쉬움 |
| 학교 예산 | 교사 자비 구매는 어렵지만 학교 예산은 가능 |
| 채널 접근성 | 어려움 — 교사 커뮤니티(인디스쿨) 직접 영업 필요 |

**잠재 수익:**
- 학교 100개 × ₩100,000 = ₩10,000,000/년
- 교사 개인 구독 200명 × $4.99/월 × 12 = $11,976/년

### Unit Economics
| 항목 | 값 |
|------|-----|
| CAC | 높음 (B2B 영업 시간) |
| LTV | 높음 (학기제 → 연간 계약) |
| 개발 공수 | 높음 (대시보드, 진도 추적 신규 개발) |

### Risks
- 1인 개발자에게 B2B 영업·지원은 시간 부담이 큼
- 서버 연동 필요 → 현재 오프라인 아키텍처와 충돌
- 학교 결제 프로세스가 복잡함 (세금계산서, 구매 결재)

### Validation Experiment
- 초등 영어 교사 10명에게 "이런 기능 있으면 쓰겠나?"라는 1시간 인터뷰
- 성공 기준: 10명 중 5명이 "학교 예산으로 구매하겠다" 응답
- 소요 시간: 4주 (인터뷰 준비 포함)

## Strategy 4: 후속 앱 생태계 (App Ecosystem)

### 설명
WordPang으로 사용자 신뢰와 브랜드 확보 → 관련 유료 앱 시리즈 출시

### How It Works
- **WordPang (무료):** 영어 단어 퍼즐 — 브랜드 허브
- **WordPang Math (유료, $1.99):** 수학 개념어 퍼즐
- **WordPang Science (유료, $1.99):** 과학 용어 퍼즐
- **WordPang 한국어 (유료, $1.99):** 초등 국어 어휘 퍼즐
- 각 앱이 WordPang 무료 앱을 통해 소개 (크로스 프로모션)

### Audience Fit
| 항목 | 평가 |
|------|------|
| 학부모 수용성 | 높음 — WordPang 신뢰 기반으로 형제 앱 구매 의향 |
| 리텐션 강화 | 강함 — 여러 앱 사용 시 이탈 비용 증가 |
| 개발 재사용성 | 높음 — 동일 아키텍처, 단어 데이터만 교체 |

**잠재 수익:**
- WordPang MAU 5,000 × 크로스 전환 10% × $1.99 = 월 $995

### Unit Economics
| 항목 | 값 |
|------|-----|
| CAC | $0 (기존 사용자) |
| LTV | $3~6 (형제 앱 1~3개 구매) |
| 개발 공수 | 낮음 (코어 재사용) |

### Risks
- WordPang이 충분한 DAU를 쌓기 전에는 형제 앱 다운로드도 낮음
- "수학 단어 퍼즐"의 가치 제안이 영어만큼 강하지 않을 수 있음

### Validation Experiment
- WordPang 리뷰·스토어 설명에 "WordPang Math 출시 예정" 링크
- 클릭률 또는 이메일 수집 수 측정 (100명+ 관심 표현 시 개발 진행)
- 소요 시간: 출시 후 90일

## Strategy 5: 학부모 프리미엄 구독 (Parent Dashboard)

### 설명
아이 앱은 무료 유지. 학부모용 진도 리포트 구독 (월 $1.99~$2.99)

### How It Works
- **아이 앱:** 무료, 변경 없음
- **학부모 대시보드 (월 $2.99):**
  - 자녀의 주간 학습 리포트 (이메일 또는 앱)
  - 카테고리별 약점 분석
  - 학원 시험 대비 추천 단어
  - 두 자녀 동시 모니터링 (계정 하나로)

### Audience Fit
| 항목 | 평가 |
|------|------|
| 학부모 지불 의향 | 중간 — 링고키즈 대비 저렴, 가시적 데이터 제공 |
| COPPA 리스크 | 주의 필요 — 자녀 학습 데이터 서버 전송 시 동의 절차 필요 |
| 개발 공수 | 높음 — 서버 연동 필요 |

### Risks
- 현재 오프라인 아키텍처 전면 변경 필요 → 핵심 차별점 약화
- 자녀 데이터 수집 → COPPA/PIPA 준수 추가 부담
- 구독 이탈 시 학부모가 아이 앱도 삭제할 수 있음

### Validation Experiment
- App Store 리뷰에서 "아이 학습 확인하고 싶다"는 언급 패턴 수집
- 10명 학부모 인터뷰: "월 $2.99에 주간 리포트 받겠나?"
- 성공 기준: 6명 이상 "예" 응답
