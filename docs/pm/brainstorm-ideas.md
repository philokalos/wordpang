# Brainstorm Ideas — WordPang (Continuous Discovery)

**Date:** 2026-03-22
**Framework:** Product Trio (PM + Designer + Engineer), Teresa Torres *Continuous Discovery Habits*
**목표:** D3 리텐션 25% → 30% (출시 후 90일)
**컨텍스트:** OST O1(첫 게임 후 재방문 동기), O3(복습 가시성), O4(streak 이탈)

---

## PM 관점 — 5가지 아이디어

### PM-1: 일일 챌린지 모드
**설명:** 매일 하나의 특별 단어 챌린지 — 해당 단어를 3회 시도 내 맞히면 특별 배지 획득

**근거:** 듀오링고의 "오늘의 수업" 방식으로 매일 방문 인센티브 제공. streak과 시너지 효과.

**핵심 가정:** 아이들이 "오늘만의 특별 챌린지"에 반응한다 (희소성 원리)

---

### PM-2: 학부모 주간 리포트 (인앱, 서버 없이)
**설명:** 매주 월요일 홈 화면에 "지난 주 요약 카드" 표시 — "지난 주 7단어 마스터! 🏆"

**근거:** 학부모 MOT #2 (유지 결정) 강화. 서버 없이 AsyncStorage 데이터만으로 구현 가능.

**핵심 가정:** 학부모가 통계 탭을 직접 들어가지 않아도 진도 확인 욕구가 있다

---

### PM-3: 카테고리별 완주 배지 (완전 수집)
**설명:** 8개 카테고리 (동물/음식/학교 등) 각각 전체 단어 달성 시 카테고리 트로피 제공

**근거:** 포켓몬 "전부 잡기" 심리 — 수집 완주 욕구가 재방문 동기 강화

**핵심 가정:** 카테고리 완주가 streak보다 장기 리텐션에 더 강력한 앵커가 된다

---

### PM-4: "오늘 단어" 예고 기능
**설명:** 전날 게임 완료 시 "내일 단어 힌트: 🐾 동물 카테고리" 예고 표시

**근거:** OST S1-C (다음 단어 미리보기) — 호기심 유발로 다음 날 방문 촉진

**핵심 가정:** 예고가 "오늘 해야 할 이유"가 된다 (FoMO 원리)

---

### PM-5: 친구 대결 모드 (비실시간)
**설명:** 같은 단어로 점수 비교 — 카카오톡 공유로 "나는 2시도에 맞혔어, 너는?" 바이럴

**근거:** 소셜 비교가 재방문 동기 강화. COPPA 이슈 없음 (데이터 서버 전송 없이 클립보드 공유)

**핵심 가정:** 초등학생이 비실시간 친구 대결에 충분한 흥미를 느낀다

---

## Designer 관점 — 5가지 아이디어

### D-1: 마스코트 캐릭터 + 감정 반응
**설명:** Sketchy 스타일 동물 마스코트가 게임 상황에 따라 다양한 표정으로 반응 (맞힘: 🎉, 틀림: 😢, 연속 맞힘: 🔥)

**근거:** 아이들은 캐릭터와 감정적 연결을 형성 → 앱 충성도 상승. 브랜드 아이덴티티 강화.

**핵심 가정:** 마스코트가 있으면 재방문율이 캐릭터 없는 버전 대비 높아진다

---

### D-2: 타일 그리기 / 색칠 채우기 시각화
**설명:** 단어를 맞힐수록 배경 격자가 채워지는 시각적 피드백 — "이번 주 50% 채웠어요!"

**근거:** 진행률 시각화가 완주 욕구 자극 (IKEA Effect). Duolingo 스트리크 불꽃 심리와 유사.

**핵심 가정:** 채워지는 격자 시각화가 숫자 통계보다 아이들에게 더 직관적으로 진도를 전달한다

---

### D-3: 결과 화면 "점프" 애니메이션
**설명:** 정답 맞혔을 때 글자 타일이 팡팡 튀어오르는 승리 애니메이션 (현재보다 더 과장되게)

**근거:** 보상 순간의 감각적 피드백 강화 → 도파민 루프 → "한 번 더 하고 싶다" (OST O1 직접 해결)

**핵심 가정:** 더 생동감 있는 승리 애니메이션이 "다시 하기" 탭률을 높인다

---

### D-4: 학습 카드 스와이프 완료 인터랙션
**설명:** 학습 카드를 왼쪽으로 스와이프 = "알았어요!", 오른쪽 스와이프 = "다시 볼게요" → 간격 반복 데이터 연동

**근거:** 수동적 읽기 → 능동적 판단 액션으로 전환. 학습 카드 완료율 향상.

**핵심 가정:** 스와이프 인터랙션이 학습 카드 참여도를 버튼 탭 방식보다 30% 높인다

---

### D-5: Streak 불꽃 성장 시각화
**설명:** streak 수에 따라 불꽃 크기·색상 변화 (1일: 작은 주황, 7일: 큰 파란 불꽃, 30일: 무지개 불꽃)

**근거:** Duolingo의 streak 불꽃보다 더 시각적으로 두드러진 성장 피드백 → streak 유지 동기 강화

**핵심 가정:** 불꽃 시각화 강화만으로도 streak 유지율이 개선된다

## Engineer 관점 — 5가지 아이디어

### E-1: 로컬 AI 단어 난이도 적응 (Adaptive Difficulty)
**설명:** 최근 10게임 정확도 통계 → 다음 게임 단어 난이도를 자동 조정 (틀릴수록 쉬운 단어, 잘 맞힐수록 어려운 단어)

**근거:** 이미 useStats의 `getDifficultyRecommendation()`이 존재. Flow 상태 (너무 쉽지도 어렵지도 않음)가 최고 참여도.

**핵심 가정:** 적응형 난이도가 고정 난이도보다 D7 리텐션을 높인다

---

### E-2: 오프라인 위젯 (iOS Lock Screen Widget)
**설명:** iOS 16+ Lock Screen Widget으로 streak 현황 + 복습 카운트 표시 — 앱 열지 않아도 인지

**근거:** 앱 아이콘 뱃지보다 더 prominent한 외부 알림 채널. 서버 없이 공유 그룹으로 데이터 전달.

**핵심 가정:** Lock Screen Widget이 앱 아이콘 뱃지보다 DAU를 더 높인다

---

### E-3: 오답 패턴 분석 + 자동 취약 단어 복습 큐
**설명:** 특정 단어를 3회 이상 틀리면 자동으로 복습 큐에 "집중 단어"로 표시 + 홈 화면 경고

**근거:** 현재 복습 큐는 시간 기반. 오답 패턴 기반 복습 → 개인화 학습 효율 향상.

**핵심 가정:** 취약 단어 집중 복습이 Words Mastered per MAU를 20% 이상 높인다

---

### E-4: 게임 결과 이미지 생성 + 공유
**설명:** 결과 화면을 이미지로 렌더링 (react-native-view-shot) → 카카오톡/인스타 공유 최적화

**근거:** 현재 텍스트 이모지 공유보다 이미지 공유가 바이럴 효과 높음. 브랜드 노출 강화.

**핵심 가정:** 이미지 공유가 텍스트 이모지 공유 대비 다운로드 전환율 2배 이상

---

### E-5: 단어 발음 음성 인식 체크 (옵션 기능)
**설명:** 마이크로 단어를 말하면 발음 정확도 체크 — 맞으면 추가 포인트

**근거:** 능동적 발음 연습 = 학부모가 원하는 교육 효과. Speaking 활동 차별화.

**핵심 가정:** 발음 음성 인식 기능이 학부모의 "교육 앱" 인식을 강화해 지속 허가 비율을 높인다
**제약:** expo-speech 인식 기능 + COPPA (마이크 권한) 복잡도 → 중장기 검토

### 상세 설명

**1위: 취약 단어 복습 큐 자동화 (E-3)**
- 현재 시간 기반 복습 + 오답 패턴 기반 집중 단어 결합
- 구현: `gameHistory` 오답 카운트 추가 → threshold(3회+) 시 `priority_review` 플래그
- 가정: Words Mastered per MAU 20% 향상

**2위: 승리 애니메이션 강화 (D-3)**
- 현재: 단순 bounce 애니메이션 → 개선: 타일 개별 jump + confetti 파티클
- 라이브러리: react-native-reanimated v3 이미 사용 중 → 추가 의존성 없음
- 가정: 결과 화면 "다시 하기" 탭률 30% → 50%

**3위: Streak 불꽃 시각화 (D-5)**
- 현재: 숫자만 표시 → 개선: streak 단계별 불꽃 이모지/색상 변화
- 구현: streak 범위 매핑 테이블 (1~3: 🔥, 4~6: 💙, 7+: 🌈)
- F1 Streak 회복 유예와 반드시 같이 구현 권장

**4위: 일일 챌린지 모드 (PM-1)**
- 매일 자정 "오늘의 단어" 고정 → 모든 사용자 동일 단어 도전
- 구현: 날짜 기반 단어 결정 알고리즘 (서버 없이 로컬 날짜 해시)
- 소셜 공유 확대: "오늘의 단어 맞힌 사람! 🏅" 공유 문화 형성

**5위: iOS Lock Screen Widget (E-2)**
- expo-widget (Expo 52 지원) 또는 react-native-widget-extension 활용
- 표시 내용: 🔥 streak 수, 📚 복습 N개 대기
- App Store 심사 시 Widget 별도 Extension 필요 → 구현 복잡도 중간
