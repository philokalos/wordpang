# Growth Loops — WordPang

**Date:** 2026-03-22
**Framework:** 5가지 성장 루프 (Ognjen Bošković)
**목표:** 유기적 성장 — $0 마케팅 예산으로 MAU 5,000 달성

---

## 현재 제품 현황

- **핵심 사용자 액션**: 단어 추론 게임 플레이 → 결과 이모지 생성
- **네트워크 효과**: 없음 (싱글 플레이어)
- **현재 공유 기능**: 결과 이모지 텍스트 클립보드 복사 (카카오톡 공유 가능)
- **기존 성장 채널**: App Store 오가닉, 맘카페 (미활성)

---

## 5가지 루프 평가

### 1. Viral Loop (바이럴 루프)

**메커니즘**: 사용자가 게임 결과를 외부 플랫폼에 공유 → 새 사용자 유입

```
게임 플레이 → 이모지 결과 생성 🟩🟧⬜ → 카카오톡/인스타 공유
→ 친구가 "이게 뭐야?" → App Store 검색 → 다운로드
```

**WordPang 적합도:** ★★★★☆ (높음)

| 항목 | 평가 |
|------|------|
| 공유 콘텐츠 | ✅ 이모지 그리드 — 짧고 호기심 유발 |
| 공유 마찰 | ⚠️ 현재 클립보드 복사 → 카카오톡 직접 공유 버튼 필요 |
| 공유 동기 | ✅ 자랑 욕구 ("2번 만에 맞혔어!"), "나도 해봐" 권유 |
| 전환율 | 중간 — 링크 없이 이모지만 공유 → "앱 이름이 뭔데?" 단계 추가 |

**루프 계수 추정:**
- 공유율: 게임 완료 사용자 중 15% 공유
- 전환율: 공유 1건당 평균 0.3명 신규 다운로드
- K-factor: 0.15 × 0.3 = **0.045** (현재) → 개선 목표 0.2+

**개선 방향:**
1. 이모지 공유에 "📱 WordPang" 브랜드 텍스트 자동 추가
2. 카카오톡 직접 공유 버튼 (KakaoSDK 연동 또는 딥링크)
3. App Store URL이 포함된 텍스트 공유 → 전환율 향상

---

### 2. Usage Loop (사용 루프)

**메커니즘**: 사용자 학습 데이터 축적 → 개인화 가치 향상 → 이탈 비용 증가

```
게임 플레이 → Words Mastered 누적 → 통계/배지 → 더 높은 개인화 가치
→ "지금 그만두면 다 사라져" → 이탈 방지 → 계속 플레이
```

**WordPang 적합도:** ★★★☆☆ (중간)

| 항목 | 평가 |
|------|------|
| 데이터 축적 | ✅ AsyncStorage에 단어 mastered 기록 저장 |
| 이탈 비용 | ⚠️ 앱 삭제 = 모든 진도 초기화 (데이터 이전 불가) |
| 가치 증가 | 중간 — 21개 배지 수집이 가장 강력한 축적 가치 |

**제약**: 완전 오프라인 → 기기 교체 시 데이터 손실 → 이탈 비용이 실질적으로 낮음

**개선 방향 (오프라인 유지):**
- 배지·통계를 QR 코드 또는 공유 가능한 "성장 카드"로 내보내기
- "지금까지 마스터한 단어 47개 — 지우기 전 저장하세요" 경고

---

### 3. Collaboration Loop (협업 루프)

**메커니즘**: 사용자가 다른 사람을 공동 경험에 초대

**WordPang 적합도:** ★★☆☆☆ (낮음)

- 현재 싱글 플레이어 앱 → 협업 루프 자연 발생 어려움
- "같은 단어 도전" 일일 챌린지는 약한 형태의 협업 루프 가능
- COPPA/개인정보 우려로 계정 기반 협업 시스템 도입 불가

**장기 가능성:** 교사 채널 (B2B2C) — 교사가 학생 클래스에 앱 도입 → 학생-학생 간 간접 협업

---

### 4. User-Generated Loop (UGC 루프)

**메커니즘**: 사용자가 콘텐츠 생성 → 다른 사용자 발견

**WordPang 적합도:** ★☆☆☆☆ (매우 낮음)

- 단어 데이터는 고정 큐레이션 → 사용자 생성 콘텐츠 없음
- App Store 리뷰가 유일한 UGC 채널이지만 플랫폼 외부

**장기 가능성:** 학부모/교사가 "커스텀 단어 리스트" 생성 공유 → v5.0+ 검토 (서버 필요)

---

### 5. Referral Loop (추천 루프)

**메커니즘**: 사용자가 인센티브를 받고 친구 초대

**WordPang 적합도:** ★★★☆☆ (중간)

| 항목 | 평가 |
|------|------|
| 추천 인센티브 | ⚠️ 무료 앱 → 금전 인센티브 없음, 배지 인센티브 가능 |
| 추천 채널 | ✅ 카카오톡, 학교 학부모 카톡방 |
| 추천 대상 | ✅ 비슷한 연령 자녀를 가진 학부모 그룹 (동질 집단) |

**구현 아이디어:**
- "친구에게 추천하면 '선생님 배지' 획득!" → 배지 인센티브 추천 루프
- 배지 달성 축하 화면에 "친구에게 알리기" CTA → 카카오톡 공유

## 추천 성장 루프 전략

### 1순위: Viral Loop 강화 (즉시 실행)

WordPang에 가장 적합하고 즉시 실행 가능한 루프.

**구현 계획:**

| 단계 | 액션 | 기간 | 지표 |
|------|------|------|------|
| 1 | 이모지 공유 텍스트에 "WordPang" + App Store 링크 추가 | 1주 | 공유 탭률 |
| 2 | 카카오톡 직접 공유 버튼 추가 (UIActivityViewController) | 2주 | 공유 완료율 |
| 3 | 배지 달성 시 자동 공유 유도 ("축하! 친구에게 자랑하기") | 3주 | K-factor |
| 4 | 일일 챌린지 결과 공유 ("오늘 단어 맞혔어! 2회 만에 🏅") | 6주 | 다운로드 전환율 |

**루프 계수 목표 진행:**
- 현재: K=0.045 (100명 → 4.5명 유기 획득)
- 1개월 후: K=0.12 (공유 버튼 개선)
- 3개월 후: K=0.25 (일일 챌린지 + 배지 공유)

---

### 2순위: Referral Loop + 맘카페 채널 (30일~)

**구현 계획:**
- 특정 배지 달성 후 "우리 아이도 해봤어요" 맘카페 후기 유도 모달
- 학부모용 "선생님 추천 배지" — 3명 친구에게 공유 시 획득
- 개발자 직접 맘카페 참여 (광고 아닌 가치 콘텐츠)

---

### 3순위: Usage Loop 강화 (60일~)

**구현 계획:**
- 배지 수집 진행률 시각화 강화 → 이탈 비용 증가
- "지금까지 마스터한 단어들" 카드 이미지 생성 → 공유 가능 포트폴리오
- 단어 Mastered 마일스톤 알림 ("50번째 단어 마스터! 🎊")

## 루프 건강 지표 (Loop Health Metrics)

| 지표 | 정의 | 목표 | 측정 방법 |
|------|------|------|---------|
| Share Rate | 게임 완료 후 공유 탭률 | 15%+ | 결과 화면 공유 버튼 탭 이벤트 |
| Share-to-Download Rate | 공유 1건당 신규 다운로드 | 0.3+ | App Store 추천인 분석 어려움 → 간접 측정 |
| K-factor | 기존 사용자 1명이 유발하는 신규 가입 | 0.25+ | (공유율) × (전환율) |
| Loop Cycle Time | 공유 → 다운로드 시간 | 24시간 내 | App Store 설치 시간 추적 |
| Referral Badge Rate | 추천 배지 획득 사용자 비율 | 5%+ | 배지 달성 이벤트 |
