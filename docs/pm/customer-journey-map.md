# Customer Journey Map — WordPang

**Date:** 2026-03-22
**페르소나:** 지은 (38세 학부모) → 지호 (9세 아들)
**JTBD:** "아이가 스스로 열고 영어 단어를 자연스럽게 익힐 수 있는 앱을 찾고 싶다"
**주목:** 이 여정은 이중 여정 — 구매 결정권자(학부모)와 실제 사용자(아이)가 다르다.

---

## Journey Map 전체 흐름

```
지은 여정:  인지 → 검토 → 설치 → 첫 사용 위임 ─────────────────────────────→ 지속 허용·추천
지호 여정:                        ↓ 앱 받음 → 온보딩 → 참여 → 습관 → 전도
```

---

## Stage 1: Awareness (인지)

**주체: 지은 (학부모)**

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 네이버 맘카페, 인스타그램 교육 계정, 유튜브 "엄마표영어" |
| **User Actions** | "초등영어앱 추천" 검색, 다른 엄마 후기 글 읽기 |
| **생각 & 질문** | "링고키즈는 애가 3주 만에 지루해했는데... 이건 다를까?" "무료라는데 진짜 광고 없나?" |
| **감정** | 😐 회의적 (이미 여러 앱 실망 경험), 🤔 궁금함 |
| **Pain Points** | 수많은 "추천" 게시물 중 진짜 좋은 앱과 광고성 글 구분 어려움 |
| **Opportunities** | 진솔한 개발자 스토리 + "왜 만들었나" 글이 신뢰도를 높임. 맘카페에 스팸 아닌 가치 글 게시 |

---

## Stage 2: Consideration (검토)

**주체: 지은 (학부모)**

| 항목 | 내용 |
|------|------|
| **Touchpoints** | App Store 제품 페이지, 스크린샷, 리뷰, 개인정보 처리방침 URL |
| **User Actions** | 스크린샷 훑어보기, 리뷰 읽기, 개인정보 탭 확인, 경쟁 앱과 빠른 비교 |
| **생각 & 질문** | "정말 광고 없나?" "개인정보 수집 없다는 게 진짜인가?" "우리 아이 수준에 맞나?" |
| **감정** | 🔍 분석적, 🛡️ 방어적 (데이터 수집 거부감 높음) |
| **Pain Points** | App Store 설명만으로는 "무데이터" 구조를 신뢰하기 어려움; 리뷰 0개 → 신뢰 없음 |
| **Opportunities** | 스크린샷 1번: 게임 장면 (아이 흥미), 마지막: 통계 화면 (학부모 신뢰). 심사 메모에 "데이터 수집 없음" 명시 |

**⚠️ Moment of Truth #1:** "이 앱이 안전하고 효과 있어 보이는가?" — Yes → 설치, No → 경쟁사로

## Stage 3: Acquisition (설치)

**주체: 지은 (학부모)**

| 항목 | 내용 |
|------|------|
| **Touchpoints** | App Store "받기" 버튼, Face ID 인증, 다운로드 |
| **User Actions** | 무료 앱 설치 (0원, 계정 불필요) |
| **생각 & 질문** | "무료니까 일단 받아보지" "계정 만들어야 하나?" |
| **감정** | 😊 가벼운 기대 (무료 + 간단한 설치) |
| **Pain Points** | 없음 — 무료·무계정·즉시 설치가 마찰 최소화 |
| **Opportunities** | 이 단계의 강점 유지. 계정 가입 절대 추가하지 말 것 |

---

## Stage 4: Onboarding — 학부모 시점

**주체: 지은 → 지호에게 폰 건네주기**

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 앱 첫 실행, 온보딩 화면, 아이에게 건네주는 순간 |
| **User Actions** | 앱 열기 → 온보딩 확인 → "야, 이거 해봐" → 아이에게 위임 |
| **생각 & 질문** | "직접 해보라고 해야지" "얼마나 어려운지 보자" |
| **감정** | 😐 관찰자 모드 |
| **Pain Points** | 온보딩이 너무 길면 학부모가 직접 끝내지 않고 아이에게 미완성으로 건넴 |
| **Opportunities** | 온보딩 < 30초 목표. 첫 화면에서 즉시 게임 시작 가능 옵션 제공 |

---

## Stage 4: Onboarding — 아이 시점

**주체: 지호 (9세)**

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 온보딩 화면, 난이도 선택, 첫 게임 |
| **User Actions** | 온보딩 읽기(또는 스킵), Easy 선택, 첫 단어 입력 시도 |
| **생각 & 질문** | "어떻게 하는 거야?" "이 색깔이 뭔 뜻이야?" "이 단어 아는데!" |
| **감정** | 😮 호기심, 😅 약간의 혼란 (색깔 의미 처음엔 모름) |
| **Pain Points** | 색깔 피드백(🟩🟧🟫) 의미를 모르면 첫 게임이 혼란스러움. 첫 단어가 너무 어려우면 즉시 포기 |
| **Opportunities** | 첫 게임에서 정답을 3~4회 안에 맞출 수 있는 쉬운 단어 배치 (CAT, DOG 등) → 첫 승리 경험 설계 |

**⭐ Aha Moment:** 첫 번째 🟩(초록) 타일이 채워지는 순간 — "아, 이게 이렇게 하는 거구나!"

## Stage 5: Engagement (참여)

**주체: 지호 (주), 지은 (간헐적 확인)**

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 게임 화면, 힌트 패널, 결과 모달, 학습 카드, 배지 알림 |
| **지호 Actions** | 게임 플레이, 힌트 전략적 사용, 배지 확인, 학습 카드 읽기 |
| **지은 Actions** | 통계 화면 주 1회 확인, "얼마나 했어?" 질문 |
| **생각 (지호)** | "이 힌트 쓰면 포인트 깎이는데 써야 하나?" "이 배지 어떻게 받아?" |
| **생각 (지은)** | "얼마나 배웠지?" "매일 하고 있나?" |
| **감정 (지호)** | 🎮 몰입, 🏆 성취감, 🤔 전략적 사고 |
| **Pain Points** | 연속으로 지면 포기하고 싶어짐; 학습 카드를 보지 않고 결과 모달만 닫아버림 |
| **Opportunities** | 패배 시 격려 메시지 + 힌트 제안; 학습 카드 자동 표시 (닫기 전 1~2초 딜레이) |

## Stage 6: Retention (유지)

**주체: 지호 (주), 지은 (허가자)**

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 배지 알림, streak 표시, 복습 큐, 일일 단어, 앱 아이콘 |
| **지호 Actions** | 매일 1게임 (streak 유지), 복습 화면 방문, 일일 단어 도전 |
| **지은 Actions** | 통계 화면으로 진도 확인 → 계속 허가 or 삭제 결정 |
| **생각 (지호)** | "오늘 streak 안 끊겨야 하는데" "이 배지 한 개만 더 받으면..." |
| **생각 (지은)** | "이번 달 47단어 학습 완료"를 보고 → "오, 효과 있네" |
| **감정 (지호)** | 🔥 streak 유지 압박(긍정), 😊 배지 수집 기쁨 |
| **Pain Points** | 복습 큐 알림 없음 → 복습 잊어버림; 여행·바쁜 날 streak 끊기면 동기 급락 |
| **Opportunities** | streak 회복 메커니즘 (1일 유예), 복습 화면 홈에서 더 prominent하게; 월간 학습 요약 통계 |

**⚠️ Moment of Truth #2 (학부모):** "통계 화면에서 성장이 보이는가?" — Yes → 계속, No → 삭제

**⚠️ Churn Trigger #1 (지호):** streak이 끊겼을 때 (특히 5일+ 이후 첫 번째 끊김)

**⚠️ Churn Trigger #2 (지은):** 아이가 앱을 3일 이상 열지 않을 때

## Stage 7: Advocacy (추천)

**주체: 지은 (주), 지호 (친구에게)**

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 공유 버튼, 이모지 결과 그리드, 맘카페, App Store 리뷰 |
| **지은 Actions** | 맘카페에 "이 앱 진짜 좋아요" 글 게시, App Store 리뷰 작성 |
| **지호 Actions** | 친구에게 "이 게임 해봐" 구두 추천, 결과 이모지 카카오톡 공유 |
| **생각** | "이 정도면 다른 엄마들한테도 알려줘야지" "우리 반 친구도 하면 같이 하겠다" |
| **감정** | 😊 뿌듯함 (좋은 것 발견), 🤝 공유 욕구 |
| **Pain Points** | 리뷰 작성 CTA가 없으면 만족해도 리뷰 안 남김; 공유 버튼 숨겨져 있으면 이모지 공유 못 함 |
| **Opportunities** | 배지 달성 후 자연스러운 "리뷰 남기기" 팝업; 결과 화면 공유 버튼 prominence 높이기 |

**⭐ Advocacy 트리거:** "우리 아이가 혼자서 영어 앱 켜요" — 이 경험이 가장 강력한 추천 소재

## Critical Moments 요약

| 순간 | 단계 | 영향 | 현재 설계 |
|------|------|------|---------|
| **Aha Moment** | 온보딩 → 첫 🟩 타일 | D1 리텐션 결정 | 있음 (개선 여지) |
| **Moment of Truth #1** | 검토 → 설치 | 다운로드 전환율 | ASO + 리뷰로 보완 필요 |
| **Moment of Truth #2** | 유지 → 장기 사용 | 학부모 허가 지속 | 통계 화면 이미 있음 |
| **Churn Trigger #1** | streak 끊김 | D7+ 이탈 | 회복 메커니즘 없음 ⚠️ |
| **Churn Trigger #2** | 3일 미방문 | 앱 삭제 | 복습 알림(뱃지) |

## 우선순위별 개선 기회

### Quick Wins (즉시 개선)
1. **첫 게임 단어 조정** — 온보딩 첫 단어를 쉬운 4글자로 고정 → 첫 승리 경험 보장 (Aha Moment 속도↑)
2. **학습 카드 자동 표시 딜레이** — 결과 모달 후 학습 카드 1.5초 자동 표시 → 완료율 개선
3. **결과 공유 버튼 prominence** — 결과 화면 상단에 공유 버튼 크게 배치

### 중기 투자 (v3.2)
4. **streak 회복 메커니즘** — "어제 하루 쉬었어도 오늘 하면 streak 유지" (1일 유예)
5. **복습 화면 홈 동선 개선** — 복습 큐 있을 때 홈 화면에 배지(뱃지) 또는 카운터 표시
6. **발음 오디오** — 학습 카드 완성도 향상 → 지은의 "효과 확인" 경험 강화

### 장기 투자
7. **학부모 월간 요약** — "이번 달 아이의 학습 리포트" 형식의 통계 요약 (이메일 없이 앱 내)
8. **streak 공유** — "🔥 10일 연속!" 공유 기능 → 친구 추천 바이럴
