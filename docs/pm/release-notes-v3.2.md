# WordPang — v3.2 Release Notes

**출시 예정일:** 2026-Q2
**버전:** 3.2.0
**대상 독자:** 학부모 + 아이 (사용자 공개용)

---

## 이번 업데이트의 핵심

WordPang v3.2는 **"더 잘 배우고, 더 오래 즐기는"** 업데이트예요. 단어 발음을 직접 들을 수 있게 됐고, streak이 끊겼을 때 다시 시작할 기회가 생겼어요. 복습할 단어가 있으면 홈 화면에서 바로 알 수 있어요.

---

## ✨ 새 기능 (New Features)

### 🔊 단어 발음 듣기

학습 카드에서 스피커 버튼(🔊)을 탭하면 해당 단어의 발음을 들을 수 있어요.

아이가 단어를 맞혔을 때 "이 단어, 어떻게 읽는 거지?"라고 궁금해한 적 있으신가요? 이제 게임이 끝난 후 학습 카드에서 바로 발음을 확인할 수 있어요. 514개 단어 모두 지원해요.

---

### 🔊 힌트에서도 발음 듣기

게임 중 발음 힌트를 사용했을 때도 스피커 버튼이 생겼어요.

텍스트로 표기된 발음 힌트에 소리까지 더했어요. 힌트를 보면서 단어를 더 잘 맞힐 수 있어요.

---

### 📚 홈 화면 복습 알림

복습할 단어가 있으면 홈 화면에 "복습 N개 준비됨" 카드가 표시돼요.

전에는 복습 탭을 직접 눌러봐야 했는데, 이제는 앱을 열자마자 오늘 복습할 단어가 몇 개인지 바로 알 수 있어요. 카드를 탭하면 복습 화면으로 바로 이동해요.

---

### ⚡ Streak 회복 기회

streak이 끊긴 다음 날, 게임을 하면 streak을 복구할 수 있어요.

하루 빠졌다고 streak이 0으로 리셋되는 건 너무 아쉬웠죠. 이제 streak이 끊긴 날 홈 화면에 "오늘 게임하면 streak 복구 가능! ⚡" 메시지가 뜨고, 그날 게임을 완료하면 streak이 돌아와요. 단, 이틀 이상 지나면 복구 불가예요.

---

### 🏆 다음 배지까지 진행률 표시

결과 화면에서 다음 배지까지 몇 게임 남았는지 볼 수 있어요.

"이 배지 어떻게 받는 거야?" 하고 궁금해했다면, 이제 게임이 끝난 후 "단어 박사까지 2게임 남았어요!" 라는 진행 표시로 다음 목표가 보여요.

---

## 🛠 개선 (Improvements)

### 연습 모드 목표 설정

연습 모드를 시작하기 전에 오늘 몇 단어를 연습할지 선택할 수 있어요 (3 / 5 / 10단어).

바쁜 날엔 3단어, 여유 있는 날엔 10단어 — 내 상황에 맞게 선택해요. 진행 중에는 상단에 "3/5 완료" 형식으로 진행 상황이 표시돼요.

---

## ♿ 접근성 (Accessibility)

### VoiceOver 지원 개선

시각 장애가 있는 사용자도 VoiceOver를 통해 게임 보드의 모든 타일 상태(정확한 위치/있지만 다른 위치/없음)를 음성으로 들을 수 있게 됐어요.

## 내부용 기술 변경 사항 (개발자 노트)

**추가된 기능:**
- `expo-speech` TTS 통합 → 학습 카드 + 발음 힌트 스피커 버튼
- `wordpang_streak_broken_date` AsyncStorage 키 추가 → 1일 streak 회복 로직
- 홈 화면 복습 큐 카운트 컴포넌트 (useFocusEffect 갱신)
- 결과 모달 배지 진행률 섹션 (achievements.ts 기반)
- 연습 모드 목표 선택 화면 (3/5/10 옵션)

**스토리 매핑:**
- S1 (발음 학습카드) → US-11 P0
- S2 (발음 힌트) → US-11 확장 P1
- S3 (홈 복습배지) → OST O3 솔루션 P0
- S4 (Streak 회복) → OST O4 솔루션 P0
- S5 (배지 진행률) → OST O1 솔루션 P1
- S6 (연습 목표) → US-14 P1
- S7 (접근성) → US-13 P2
