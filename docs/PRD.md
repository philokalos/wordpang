## 1. Executive Summary

WordPang은 초등학생(7~13세)을 위한 iOS 영어 단어 학습 게임이다. 색깔 힌트 기반 글자 추론 메커니즘에 교정 피드백(+494% 기억력), 간격 반복, 적응형 난이도, 역량 동기 이론을 결합하여 기존 단순 암기 앱과 차별화된다. 완전 오프라인·무광고·무데이터 수집 구조로 개인정보 리스크 없이 초등학생에게 안전하게 제공 가능하다.

## 2. Background & Context

### 문제 공간

초등학생 영어 단어 학습 시장에는 두 가지 공백이 있다.

1. **흥미 공백**: 플래시카드·단순 퀴즈 앱은 반복 암기로 인해 흥미 유지율이 낮다.
2. **교육 품질 공백**: 게임 요소가 있는 앱은 교육 연구 기반이 부족하고, 연구 기반 앱은 지루하다.

### 교육 연구 기반

| 전략 | 연구 근거 | WordPang 적용 |
|------|----------|--------------|
| 교정 피드백 | +494% 기억력 향상 (Butler et al.) | 학습 카드 (게임 종료 후 피드백) |
| 간격 반복 | +53% 어휘 획득률 (Nakata, 2015) | 복습 시스템 (30분 × 2ⁿ) |
| 적응형 난이도 | Flow Zone Theory (Csikszentmihalyi) | 승률 기반 난이도 추천 |
| 역량 동기 | Competence Motivation Theory (Harter) | 21개 업적 배지 시스템 |

### 제품 이력

| 버전 | 내용 |
|------|------|
| v1.x | Wordle 클론 프로토타입 (영어 사전 단어) |
| v2.x | 초등 단어 특화, 난이도 분리 |
| v3.0 | Expo 52 리빌드, 카테고리·복습·업적 시스템 (완료) |
| v3.1 | Sketchy 손그림 디자인 시스템 (완료) |
| **v3.2** | **발음 오디오, 단어 확장, 접근성 강화 (계획)** |

### 시장 배경

- 국내 초등 영어 학습 앱은 대부분 구독 기반(매월 결제) 또는 광고 기반
- WordPang: 일회성 유료 또는 무료 제공으로 단순한 가치 교환
- 개인정보 수집 없음 → COPPA·PIPA 완전 준수, 학교·학부모 채택 장벽 최소

## 3. Objectives & Success Metrics

### Goals

1. App Store에서 초등 영어 교육 카테고리 상위 50위 진입 (출시 후 90일 내)
2. DAU/MAU 비율 20% 이상 유지 — 매일 돌아오는 습관 형성
3. 평균 사용자가 첫 달에 50개+ 단어를 학습 완료 상태로 전환

### Non-Goals

다음은 현재 범위에서 제외된다. (이유 포함)

| 제외 항목 | 제외 이유 |
|----------|---------|
| 서버 연동 / 클라우드 동기화 | 개인정보 수집 없는 오프라인 정책 유지 |
| 멀티플레이어 / 랭킹 | 아동 소셜 기능 COPPA 리스크 |
| Android 지원 | 단일 플랫폼 품질 집중 (iOS 먼저 검증) |
| 음성 인식 발음 연습 | 마이크 권한 → 학부모 동의 복잡성 증가 |
| 사용자 계정 / 로그인 | 개인정보 수집 없는 정책과 충돌 |
| Push Notification | 아동 앱 알림 규제 및 배터리 영향 |

## 4. Target Users & Segments

### 주 사용자: 초등학생 (7~13세)

- **학습 단계**: 영어 학습 초기(알파벳 수준) ~ 중급(기초 문장 수준)
- **특성**: 모바일 게임 친숙, 짧은 집중력(5~15분), 시각적 피드백에 반응
- **동기**: 게임 클리어, 배지 수집, 친구와 비교
- **현재 대안**: 학원 단어 시험, 단어장 암기, 교과서 부록

### 보조 사용자: 학부모

- **역할**: 앱 구매 결정권자, 학습 진도 모니터링
- **주요 관심사**: 교육 효과 가시성, 개인정보 안전, 광고 없음
- **접점**: 통계 화면 (학습 진도 확인)

### 사용 시나리오

| 시나리오 | 빈도 | 특성 |
|---------|------|------|
| 등·하교 버스 (5~10분) | 매일 | 짧은 세션, 연습 모드 적합 |
| 취침 전 일과 (10~15분) | 매일 | 일일 단어 + 복습 |
| 주말 자유 시간 (15~30분) | 주 2회 | 전체 게임 + 업적 탐색 |
| 학부모 모니터링 | 주 1회 | 통계 화면 |

## 6. Solution Overview

### 아키텍처 원칙

1. **완전 오프라인**: 모든 데이터는 AsyncStorage에만 저장. 서버·네트워크 요청 없음.
2. **교육 연구 기반**: 4가지 검증된 교육 전략을 게임 메커니즘에 직접 매핑.
3. **아동 친화 UX**: Sketchy 손그림 디자인 시스템 — Gaegu 폰트, 크레용 팔레트, 미세 회전.
4. **성능 우선**: 고빈도 컴포넌트(LetterTile, Keyboard)는 SVG 없이 60fps 유지.

### 기술 스택

| 레이어 | 기술 |
|------|------|
| 플랫폼 | Expo SDK 52 + React Native 0.76 |
| 언어 | TypeScript (strict mode) |
| 라우팅 | expo-router (파일 기반) |
| 애니메이션 | React Native Reanimated |
| 사운드 | expo-av |
| 햅틱 | expo-haptics |
| 저장 | AsyncStorage (`wordpang_` 접두사) |
| 테스트 | Jest + React Native Testing Library |
| 빌드 | EAS Build → TestFlight → App Store |

### 핵심 알고리즘

**2-Pass 추측 평가** (useGame 훅):
- Pass 1: 정확한 위치(correct) 먼저 표시
- Pass 2: 남은 글자에서 위치 불일치(present) 판정
- 중복 글자 처리: 정답 글자 수 이상으로 present 표시 안 함

**간격 반복 스케줄** (spaced-repetition.ts):
- 간격: 30분 × 2ⁿ (n = 복습 횟수)
- 상태: `new` → `learning` (5회+) → `mastered`

### 화면 구조

```
app/
├── index.tsx       홈 (난이도/카테고리 선택, 일일 모드, 통계 요약)
├── game.tsx        게임 (게임 보드, 키보드, 힌트, 결과 모달)
├── stats.tsx       통계 (승률, 추측 분포, 난이도별 통계)
├── review.tsx      복습 (플래시카드, 간격 반복 큐)
├── practice.tsx    연습 (집중 세션, 세션 요약)
└── onboarding.tsx  온보딩 (첫 실행 가이드)
```
