# Product Requirements Document — WordPop v3.0

## 1. 제품 개요

**제품명:** WordPop
**버전:** 3.0.0
**플랫폼:** iOS (iPhone 전용)
**대상 사용자:** 초등학생 (7~13세)
**핵심 가치:** 게임 기반 영어 단어 학습으로 자기주도 어휘력 향상
**번들 ID:** com.philokalos.wordpop

## 2. 문제 정의

### 배경
초등학생이 영어 단어를 학습할 때 반복 암기 방식은 흥미를 잃기 쉽고, 장기 기억으로의 전환율이 낮다. 기존 단어 학습 앱은 성인 대상이거나 교육 연구 기반이 부족한 경우가 많다.

### 해결하려는 문제
- 초등학생 눈높이에 맞는 영어 단어 학습 도구 부재
- 단순 암기 → 장기 기억 전환의 어려움
- 학습 동기 유지의 어려움

### 교육 연구 근거
| 전략 | 연구 근거 | 적용 기능 |
|------|----------|----------|
| 교정 피드백 | 494% 기억력 향상 (Butler et al.) | 학습 카드 |
| 간격 반복 | 53% 어휘 획득률 (Nakata, 2015) | 복습 시스템 |
| 적응형 난이도 | Flow Zone Theory (Csikszentmihalyi) | 난이도 제안 |
| 역량 동기 | Competence Motivation Theory (Harter) | 업적 배지 |

## 3. 목표 사용자

### 주요 사용자: 초등학생 (7~13세)
- 영어 학습 초기~중급 단계
- 모바일 기기 사용에 익숙
- 게임 형태의 학습에 높은 참여도

### 보조 사용자: 학부모
- 자녀의 학습 진도 확인 (통계 화면)
- 교육적 가치가 있는 앱 선호

## 4. 핵심 기능 (Feature Set)

### 4.1 게임 플레이
| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 단어 맞추기 | Wordle 방식의 글자 추론 게임 | P0 |
| 3단계 난이도 | Easy(4자), Normal(5자), Hard(6자/7회) | P0 |
| 8가지 카테고리 | animal, food, school, nature, body, home, action, feeling | P0 |
| 색상 피드백 | correct(초록), present(주황), absent(갈색) | P0 |
| 키보드 상태 | 사용한 글자의 상태를 키보드에 반영 | P0 |

### 4.2 힌트 시스템
| 힌트 유형 | 비용 | 설명 |
|----------|------|------|
| 예시 문장 (example) | 1pt | 단어가 포함된 영어 문장 |
| 첫 글자 (firstLetter) | 1pt | 정답의 첫 번째 글자 |
| 모음 수 (vowelCount) | 1pt | 단어에 포함된 모음 개수 |
| 한국어 뜻 (meaning) | 1pt | 단어의 한국어 의미 |
| 발음 (pronunciation) | 1pt | 로마자 발음 표기 |
| 라이밍 (rhyming) | 1pt | 운율이 맞는 단어 |
| 글자 위치 (letterPosition) | 2pt | 특정 위치의 글자 공개 |
| 단어 패밀리 (wordFamily) | 2pt | 같은 어근의 단어들 |

- 게임당 최대 4포인트 사용 가능

### 4.3 학습 기능
| 기능 | 설명 |
|------|------|
| 학습 카드 | 게임 종료 후 정답 단어의 뜻, 발음, 예문 표시 |
| 간격 반복 복습 | 30분 × 2^n 간격으로 복습 알림 |
| 복습 상태 | new → learning(5회+) → mastered |
| 연습 모드 | 5~10분 집중 학습 세션 |

### 4.4 동기 부여
| 기능 | 설명 |
|------|------|
| 21개 업적 배지 | 다양한 조건 달성 시 배지 해금 |
| 연속 기록 (streak) | 연속 플레이 일수 추적 |
| 일일 단어 | 매일 1개 새로운 단어 도전 |
| 통계 대시보드 | 승률, 추측 분포, 카테고리 진도 |

### 4.5 적응형 시스템
| 기능 | 설명 |
|------|------|
| 난이도 제안 | 승률 기반 자동 난이도 추천 |
| 카테고리 진도 | 카테고리별 학습 완료율 표시 |

## 5. 화면 구성

| 화면 | 경로 | 주요 기능 |
|------|------|----------|
| 홈 | `/` | 난이도/카테고리 선택, 일일 모드, 통계 요약 |
| 게임 | `/game` | 게임 보드, 키보드, 힌트, 결과 모달 |
| 통계 | `/stats` | 승률, 추측 분포, 난이도별 통계 |
| 복습 | `/review` | 플래시카드, 복습 큐, 진도 |
| 연습 | `/practice` | 집중 세션, 세션 요약 |
| 온보딩 | `/onboarding` | 첫 실행 가이드 |

## 6. 데이터 모델

### 단어 데이터 (360개)
- 난이도별 120개 (easy/normal/hard)
- 8개 카테고리 균등 배분
- 각 단어: word, meaning(한국어), pronunciation, example, category, partOfSpeech

### 로컬 저장 (AsyncStorage)
| 키 | 데이터 |
|-----|--------|
| wordpop_stats | 총 플레이, 승리, 연속기록, 추측 분포 |
| wordpop_daily_state | 일일 모드 상태 |
| wordpop_learned_words | 학습 완료 단어 목록 |
| wordpop_review_entries | 복습 스케줄 (간격 반복) |
| wordpop_achievements | 해금된 업적 |
| wordpop_played_categories | 플레이한 카테고리 |

## 7. 디자인 시스템 — Sketchy

### 컨셉
초등학생 친화적인 손그림/크레용 스타일 UI

### 핵심 요소
| 요소 | 설명 |
|------|------|
| 폰트 | Gaegu (한국어+영어 손글씨체) |
| 테두리 | 불규칙한 SVG 손그림 테두리 (SketchyBox) |
| 배경 | 도트 그리드 노트 배경 (PaperBackground) |
| 색상 | 크레용 팔레트 (green, orange, red, blue, purple, yellow, pink) |
| 회전 | 미세한 랜덤 기울기 (±1.5도) |
| 장식 | 별, 구불구불 선 (DoodleDecoration) |

### 성능 고려
- 고빈도 컴포넌트 (LetterTile, Keyboard): 스타일 변경만, SVG 없음 → 60fps 유지
- 저빈도 컨테이너 (모달, 카드): SketchyBox/SketchyButton 래퍼 사용

## 9. 제약 사항 및 미포함 기능

### 현재 미포함
- 서버 연동 / 클라우드 동기화
- 멀티 플레이어
- Android 지원
- 태블릿 최적화 레이아웃
- 알림 (Push Notification)
- 음성 인식 발음 연습
- 사용자 계정 / 로그인

### 기술적 제약
- Expo SDK 52 버전 동결
- AsyncStorage 용량 제한 (일반적으로 6MB)
- 오프라인 전용 → 데이터 백업은 수동

## 10. 성공 지표

| 지표 | 목표 |
|------|------|
| 일일 활성 사용 | 학습 세션 완료율 > 70% |
