## 1. 게임 플레이 (Core Gameplay)

### TS-01: 정상 게임 플로우 — 승리
**우선순위:** Critical
**전제조건:** 앱 설치 완료, Normal 난이도 선택

| Step | Action | Expected Result |
|------|--------|----------------|
| 1 | 홈에서 Normal 난이도 선택 → 게임 시작 | 5칸 × 6행 게임 보드 표시 |
| 2 | 키보드로 5글자 단어 입력 | 입력 칸에 글자 표시, 팝 애니메이션 |
| 3 | Enter 제출 | 타일 뒤집기 애니메이션 후 🟩🟧⬜ 색상 표시 |
| 4 | 키보드 해당 키에 색상 반영 확인 | correct=초록, present=주황, absent=갈색 |
| 5 | 정답 단어 입력 후 제출 | 모든 타일 초록, 바운스 애니메이션 |
| 6 | 결과 모달 확인 | "정답!" 메시지, 학습 카드(뜻·발음·예문·품사) 표시 |
| 7 | 통계 업데이트 확인 | 플레이 횟수+1, 승리+1, 추측 분포 업데이트 |

**사후조건:** 단어가 learned_words에 추가됨, review_entries에 추가됨
**우선순위:** Critical

### TS-19: 간격 반복 스케줄 검증
**우선순위:** High
**목적:** 30min × 2^n 알고리즘 검증

| reviewCount | 예상 nextReview 간격 |
|-------------|---------------------|
| 0 → 1 | 30분 (30 × 2^0) |
| 1 → 2 | 60분 (30 × 2^1) |
| 2 → 3 | 120분 (30 × 2^2) |
| 3 → 4 | 240분 (30 × 2^3) |

**검증 방법:** 유닛 테스트 또는 AsyncStorage 직접 확인
**우선순위:** High

---
