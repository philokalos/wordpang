# WordPop 사용자 가이드

## WordPop이란?

WordPop은 초등학생을 위한 영어 단어 학습 게임 앱입니다. Wordle 방식으로 영어 단어를 추측하면서 자연스럽게 단어를 익힐 수 있어요.

---

## 게임 방법

### 1. 난이도 선택

홈 화면에서 원하는 난이도를 선택하세요.

| 난이도 | 글자 수 | 시도 횟수 |
|--------|---------|----------|
| Easy 🌟 | 4글자 | 6번 |
| Normal ⭐ | 5글자 | 6번 |
| Hard 💪 | 6글자 | 7번 |

### 2. 카테고리 선택

8가지 카테고리 중 원하는 주제를 선택할 수 있어요.

| 카테고리 | 예시 단어 |
|----------|----------|
| 🐾 동물 (animal) | BEAR, DUCK, FISH |
| 🍽️ 음식 (food) | CAKE, RICE, MILK |
| 🏫 학교 (school) | BOOK, DESK, MATH |
| 🌿 자연 (nature) | TREE, RAIN, STAR |
| 🦴 신체 (body) | HAND, FACE, NOSE |
| 🏠 집 (home) | DOOR, LAMP, SOFA |
| 🏃 동작 (action) | JUMP, WALK, SING |
| 💛 감정 (feeling) | GLAD, HOPE, CALM |

### 3. 단어 입력

- 키보드를 사용해 영어 단어를 입력하세요
- 입력 후 Enter 키를 누르면 결과가 표시됩니다

### 4. 색상 피드백 이해하기

| 색상 | 의미 |
|------|------|
| 🟩 초록 | 정확한 위치에 올바른 글자 |
| 🟧 주황 | 단어에 포함되지만 위치가 다름 |
| 🟫 갈색 | 단어에 포함되지 않은 글자 |

---

## 힌트 사용하기

게임 중 어려울 때 힌트를 사용할 수 있어요. 게임당 최대 **4포인트**까지 사용 가능합니다.

| 힌트 | 비용 | 알려주는 것 |
|------|------|------------|
| 예시 문장 | 1pt | 단어가 들어간 영어 문장 |
| 첫 글자 | 1pt | 정답의 첫 번째 알파벳 |
| 모음 수 | 1pt | 단어의 모음(a,e,i,o,u) 개수 |
| 한국어 뜻 | 1pt | 단어의 한국어 의미 |
| 발음 | 1pt | 소리나는 대로 적은 발음 |
| 라이밍 | 1pt | 끝소리가 같은 단어 |
| 글자 위치 | 2pt | 특정 위치의 글자 공개 |
| 단어 패밀리 | 2pt | 같은 어근의 단어들 |

## 학습 카드

게임이 끝나면 정답 단어의 **학습 카드**가 나타나요.

카드에는 다음 정보가 표시됩니다:
- **단어** (영어)
- **의미** (한국어 뜻)
- **발음** (읽는 법)
- **예문** (영어 문장)
- **카테고리** (주제)
- **품사** (명사, 동사, 형용사 등)

이 카드를 통해 단어를 더 깊이 이해할 수 있어요.

---

## 복습 모드

### 간격 반복 학습

학습한 단어는 자동으로 복습 목록에 추가됩니다. 과학적으로 검증된 **간격 반복** 방식으로 복습 시간을 알려줘요.

| 복습 횟수 | 다음 복습까지 |
|----------|-------------|
| 1회 | 30분 |
| 2회 | 1시간 |
| 3회 | 2시간 |
| 4회 | 4시간 |
| 5회 | 8시간 |
| 6회+ | 계속 2배씩 |

### 복습 상태

| 상태 | 의미 |
|------|------|
| 🆕 new | 아직 복습하지 않은 단어 |
| 📖 learning | 복습 중인 단어 (5회 미만) |
| ⭐ mastered | 충분히 복습된 단어 (5회 이상) |

---

## 연습 모드

집중적으로 단어를 연습하고 싶을 때 사용하세요.

- **5~10분** 집중 세션
- 세션 종료 후 결과 요약 제공
- 학습한 단어 수, 정답률 확인

---

## 일일 단어 모드

매일 새로운 단어가 출제됩니다. 홈 화면에서 일일 모드를 켜고 도전하세요.

---

## 통계

통계 화면에서 학습 진도를 확인할 수 있어요.

- **총 플레이 횟수** / **승리 횟수**
- **현재 연속 기록** / **최고 연속 기록**
- **추측 분포** (몇 번째에 맞혔는지 그래프)
- **난이도별 통계**
- **카테고리별 진도**

## 팁

1. **매일 한 번씩** 플레이하면 연속 기록이 올라가요
2. **힌트를 아끼면** "힌트 없이!" 배지를 받을 수 있어요
3. **다양한 카테고리**를 플레이하면 "탐험가" 배지를 받아요
4. **복습 알림**이 있을 때 바로 복습하면 기억에 더 잘 남아요
5. Easy부터 시작해서 점점 어려운 단계에 도전해 보세요
